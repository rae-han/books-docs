# Chapter 4: Anatomy of a Flashlight (손전등의 해부)

## 핵심 질문

> 손전등은 어떻게 작동하는가? 전기의 본질은 무엇이며, 전자가 흐르는 완전한 경로인 "회로"는 왜 필수적인가?

---

## 1. 손전등 분해하기

Chapter 1에서 두 아이는 손전등으로 대화했다. 이제 그 손전등의 내부를 들여다본다. 가장 단순한 손전등의 구성 요소:

| 부품 | 역할 |
|------|------|
| **배터리(battery)** | 전기 에너지의 원천 |
| **전구(bulb)** 또는 LED | 전기 에너지를 빛으로 변환 |
| **스위치(switch)** | 회로를 연결하거나 끊음 |
| **금속 접점/전선** | 전기가 흐르는 경로 |
| **케이스** | 부품을 고정하는 구조물 |

이 단순한 장치 안에 전기의 모든 기본 원리가 담겨 있다.

---

## 2. 원자의 세계 — 전기의 근원

전기를 이해하려면 물질의 가장 작은 단위인 **원자(atom)**에서 시작해야 한다.

### 2.1 원자의 구조

```
        전자(electron) ⊖
           ○ ← 궤도를 돌고 있음
          /
    ┌─────────┐
    │ ⊕⊕⊕ ○○○ │ ← 원자핵(nucleus)
    │ 양성자  중성자│   양성자(proton): 양전하(+)
    └─────────┘   중성자(neutron): 전하 없음
          \
           ○
        전자(electron) ⊖ ← 음전하(-)
```

- **양성자(proton)**: 양전하(+), 원자핵 안에 있음
- **중성자(neutron)**: 전하 없음, 원자핵 안에 있음
- **전자(electron)**: 음전하(-), 원자핵 주위를 돎

정상 상태의 원자에서는 양성자 수 = 전자 수이므로 전체 전하는 0이다.

### 2.2 전자의 특별한 성질

전기에서 핵심 역할을 하는 것은 **전자**다. 특히 원자핵에서 가장 먼 궤도(최외각)의 전자가 중요하다. 어떤 원소의 최외각 전자는 원자핵에 느슨하게 묶여 있어서 쉽게 이탈할 수 있다.

> **Petzold의 비유**: 최외각 전자는 마치 도시 외곽에 사는 주민과 같다. 도심(원자핵)의 인력이 약해서 쉽게 옆 도시(옆 원자)로 이사할 수 있다.

---

## 3. 도체와 절연체

### 3.1 도체(Conductor)

최외각 전자가 쉽게 이동할 수 있는 물질을 **도체**라 한다. 대표적인 도체:

- **금속**: 구리(Cu), 은(Ag), 금(Au), 알루미늄(Al)
- 특징: 최외각 전자가 **자유 전자(free electron)**로서 원자 사이를 자유롭게 돌아다님

구리 전선 안에서는 수십억 개의 자유 전자가 무작위로 이동하고 있다. 아직 특정 방향으로 흐르지는 않는다 — 배터리가 연결되어야 한다.

### 3.2 절연체(Insulator)

전자가 원자에 단단히 묶여 있어 이동하기 어려운 물질:

- **고무, 플라스틱, 유리, 세라믹, 공기**
- 전선의 피복이 고무나 플라스틱인 이유: 전류가 엉뚱한 곳으로 새지 않게

| 구분 | 도체 | 절연체 |
|------|------|--------|
| 자유 전자 | 많음 | 거의 없음 |
| 전기 흐름 | 용이 | 극히 어려움 |
| 예시 | 구리, 은, 금 | 고무, 유리, 플라스틱 |
| 용도 | 전선, 회로 | 전선 피복, 절연 |

> **핵심 통찰**: 도체와 절연체의 구분은 **자유 전자의 수**에 의해 결정된다. 이 스펙트럼의 중간에 있는 물질이 **반도체(semiconductor)**이며, Chapter 15에서 트랜지스터의 핵심 소재로 등장한다.

## 4. 전압, 전류, 저항 — 전기의 세 가지 얼굴

### 4.1 전압(Voltage)

**전압**은 전자를 밀어내는 "압력"이다. 단위는 **볼트(Volt, V)**.

> **Petzold의 비유**: 전압은 수도관의 **수압**과 같다. 수압이 높으면 물이 세게 나오듯, 전압이 높으면 전자를 더 강하게 밀어낸다. 배터리는 일종의 펌프로, 화학 반응을 통해 전자에 에너지를 부여한다.

배터리의 두 단자:
- **양극(+)**: 전자가 부족한 쪽 → 전자를 끌어당김
- **음극(-)**: 전자가 풍부한 쪽 → 전자를 밀어냄

일반 손전등 배터리(AA): 1.5V

### 4.2 전류(Current)

**전류**는 전자의 흐름량이다. 단위는 **암페어(Ampere, A)**.

> **Petzold의 비유**: 전류는 수도관을 흐르는 **물의 유량**과 같다. 초당 얼마나 많은 물(전자)이 흘러가는가?

기술적으로, 1암페어 = 초당 약 6.24 × 10¹⁸ 개의 전자가 지나가는 양이다. 엄청난 수지만, 전자는 극히 작으므로 가능하다.

**관례적 전류 방향**: 역사적 이유로 전류의 방향은 +에서 -로 향한다고 약속한다. 실제 전자는 -에서 +로 흐르지만, 회로 분석에서는 관례적 방향을 사용한다.

### 4.3 저항(Resistance)

**저항**은 전류의 흐름을 방해하는 정도다. 단위는 **옴(Ohm, Ω)**.

모든 물질은 어느 정도의 저항을 가진다. 도체도 저항이 0은 아니며, 전선이 길어질수록 저항이 커진다.

### 4.4 옴의 법칙(Ohm's Law)

전압, 전류, 저항의 관계:

```
V = I × R

V: 전압(볼트)
I: 전류(암페어)
R: 저항(옴)
```

| 비유 | 전기 | 수도관 |
|------|------|--------|
| 전압(V) | 전자를 미는 힘 | 수압 |
| 전류(I) | 전자의 흐름량 | 유량 |
| 저항(R) | 흐름을 방해하는 정도 | 관의 좁기 |

수압이 높고 관이 넓으면 물이 많이 흐르듯, 전압이 높고 저항이 낮으면 전류가 많이 흐른다.

## 5. 회로 — 전기의 완전한 경로

### 5.1 왜 "회로(circuit)"인가?

"Circuit"은 "circle(원)"에서 유래한다. 전류가 흐르려면 전자가 출발점(배터리 -)에서 나와서 일을 하고(전구를 밝히고) 다시 출발점(배터리 +)으로 돌아오는 **완전한 순환 경로**가 있어야 한다.

```
      ┌──── 스위치 ────┐
      │    (열림/닫힘)    │
  ┌───┴───┐         ┌───┴───┐
  │       │         │       │
  │ 배터리 │         │  전구  │
  │  1.5V │         │       │
  │       │         │       │
  └───┬───┘         └───┬───┘
      │                 │
      └─────────────────┘
         (전선으로 연결)
```

- **스위치 닫힘(closed)**: 경로 완성 → 전류 흐름 → 전구 켜짐
- **스위치 열림(open)**: 경로 끊김 → 전류 차단 → 전구 꺼짐

### 5.2 단선과 합선

| 상태 | 설명 | 결과 |
|------|------|------|
| 정상 회로 | 배터리 → 전구 → 배터리 완전한 경로 | 전구 밝게 빛남 |
| **단선(open circuit)** | 경로가 어딘가에서 끊김 | 전류 없음, 전구 꺼짐 |
| **합선/단락(short circuit)** | 전구를 거치지 않고 직접 연결 | 과전류 → 배터리 과열 → 위험! |

합선은 저항이 거의 0인 경로로 전류가 흐르는 것이다. 옴의 법칙에 의해 R이 0에 가까우면 I가 무한대에 가까워진다 — 배터리가 과열되거나 전선이 녹을 수 있다.

---

## 6. 회로도의 기호

실제 회로를 그릴 때는 사실적인 그림 대신 **회로도(circuit diagram/schematic)** 기호를 사용한다:

```
배터리:          스위치(열림):      스위치(닫힘):
  ─┤├─            ─/ ─              ─/─
  긴선=+          (끊어진 선)       (연결된 선)
  짧선=-

전구/저항:       전선 교차(연결 없음):  전선 연결:
  ─/\/\/─          ─┼─                    ─●─
                 (그냥 지나감)          (점으로 표시)

접지(ground):
  ─┤
   ≡
```

### 간단한 손전등 회로도

```
        S (스위치)
    ┌───/ ───┐
    │        │
   ─┤├─     [R] (전구)
   배터리    │
    │        │
    └────────┘
```

이 간단한 회로도가 앞으로 수백 페이지에 걸쳐 점점 복잡해지지만, 기본 원리는 동일하다 — 전원, 경로, 부하(일을 하는 부품), 스위치.

## 7. 배터리의 원리

배터리는 **화학 에너지를 전기 에너지로 변환**하는 장치다.

- 두 가지 다른 금속(또는 화합물)을 전해질에 담금
- 화학 반응으로 한쪽에 전자가 쌓이고(음극), 다른 쪽에서 전자가 빠져나감(양극)
- 외부 회로를 통해 전자가 음극 → 양극으로 이동 = 전류

직렬 연결로 전압을 높일 수 있다:
- AA 배터리 1개 = 1.5V
- AA 배터리 2개 직렬 = 3.0V
- AA 배터리 4개 직렬 = 6.0V (일반 손전등)

> **핵심 통찰**: 배터리는 전자를 "밀어내는 펌프"이고, 전구는 전자의 에너지를 "소비하는 부하"이며, 전선은 전자가 이동하는 "길"이다. 이 세 요소(전원-경로-부하)는 가장 복잡한 컴퓨터 회로에서도 변하지 않는 기본 구조다.

## 회로 구성

손전등 회로의 전자 흐름을 단계별로 추적하면:

```
1. 배터리 음극(-): 화학 반응으로 전자가 축적됨
         ↓
2. 전선: 전자가 도체(금속)를 따라 이동
         ↓
3. 스위치: 닫혀 있으면 통과, 열려 있으면 멈춤
         ↓
4. 전구의 필라멘트: 얇은 텅스텐 → 높은 저항 → 전자가 에너지를 잃으며 열과 빛 발생
         ↓
5. 전선: 에너지를 잃은 전자가 계속 이동
         ↓
6. 배터리 양극(+): 전자가 돌아옴 → 화학 반응이 다시 에너지를 공급
         ↓
(1로 반복)
```

---

## 요약

- 손전등은 **배터리, 전구, 스위치, 전선**으로 구성된 가장 단순한 전기 회로다
- **원자**는 양성자(+), 중성자, 전자(-)로 구성되며, 전기는 **전자의 흐름**이다
- **도체**는 자유 전자가 많아 전기가 잘 통하고, **절연체**는 그렇지 않다
- **전압**(V)은 전자를 미는 힘, **전류**(I)은 흐름량, **저항**(R)은 방해 정도이며, V = I × R 관계를 가진다
- **회로**는 전기가 흐를 수 있는 완전한 순환 경로이며, 끊기면 전류가 흐르지 않는다
- 회로도 기호를 사용하여 회로를 추상적으로 표현할 수 있다
