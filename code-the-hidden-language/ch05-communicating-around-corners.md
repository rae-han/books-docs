# Chapter 5: Communicating Around Corners (모퉁이 너머로 통신하기)

## 핵심 질문

> 빛이 아닌 전선을 사용하면 통신의 한계를 어떻게 극복할 수 있으며, 양방향 통신과 장거리 통신에서 발생하는 문제들은 무엇인가?

---

## 1. 손전등의 한계

Chapter 1의 두 아이는 손전등으로 통신했다. 하지만 이 방법에는 근본적인 한계가 있다:

- **직선 시야(line of sight)** 필요: 두 사람 사이에 장애물이 있으면 불가능
- **날씨와 환경**: 안개, 비, 강한 햇빛에서는 사용 어려움
- **거리**: 멀어질수록 빛이 약해짐
- **방향**: 모퉁이를 돌아갈 수 없음

해결책: 빛 대신 **전선(wire)**을 사용하자. 전선은 모퉁이를 돌 수 있고, 벽을 통과할 수 있으며, 날씨에 영향을 받지 않는다.

---

## 2. 전선을 이용한 기본 통신 회로

Chapter 4에서 배운 회로 지식을 적용한다. 한 집에 배터리와 스위치를, 다른 집에 전구를 놓고 전선으로 연결한다:

```
[집 A: 보내는 사람]              [집 B: 받는 사람]
                전선 1
    ┌──── 스위치 ──────────────────── 전구 ────┐
    │                                          │
   ─┤├─ 배터리                                  │
    │                                          │
    └──────────────────────────────────────────┘
                전선 2
```

- 집 A에서 스위치를 누르면 → 회로 완성 → 집 B의 전구가 켜짐
- 스위치를 떼면 → 회로 끊김 → 전구가 꺼짐
- 모스 부호를 사용하여 메시지 전송 가능!

**필요한 전선: 2개** (가는 길 + 오는 길 — 전류가 순환해야 하므로)

---

## 3. 양방향 통신의 문제

위 회로는 **단방향(simplex)**이다. A는 B에게 보낼 수 있지만, B는 A에게 보낼 수 없다.

양방향 통신을 위해서는 **반대 방향**으로도 같은 장치가 필요하다:

```
[집 A]                              [집 B]
         전선 1 (A→B)
  스위치A ──────────────────── 전구B
  배터리A

  전구A ──────────────────── 스위치B
         전선 2 (B→A)          배터리B

  (공통 귀환 전선?)
         전선 3
  ───────────────────────────────
```

단순하게 구성하면 **전선 4개**가 필요하다:
- A→B 방향: 전선 2개 (가는 길 + 오는 길)
- B→A 방향: 전선 2개 (가는 길 + 오는 길)

두 집 사이에 전선 4개를 설치하는 것은 비용과 노력이 크다. 줄일 수 있을까?

---

## 4. 공통 회선으로 전선 줄이기

핵심 관찰: 두 회로 모두 "돌아오는 전선"이 필요하다. 이 **귀환 전선(return wire)**을 공유하면 전선 하나를 절약할 수 있다!

```
[집 A]                              [집 B]
         전선 1 (A→B 신호)
  스위치A ──────────────────── 전구B
  배터리A                     배터리B
  전구A ──────────────────── 스위치B
         전선 2 (B→A 신호)
    │                              │
    └──────────────────────────────┘
         전선 3 (공통 귀환선)
```

**전선 3개**로 양방향 통신이 가능하다. 1개를 절약했다.

하지만 더 줄일 수 있다.

---

## 5. 접지(Ground) — 지구를 전선으로 사용하기

### 5.1 발견

초기 전신 기술자들은 놀라운 발견을 했다: **귀환 전선을 아예 없앨 수 있다.** 양쪽 끝의 전선을 땅에 꽂으면, 지구 자체가 귀환 경로 역할을 한다!

```
[집 A]                              [집 B]
         전선 1 (A→B 신호)
  스위치A ──────────────────── 전구B
  배터리A                     배터리B
  전구A ──────────────────── 스위치B
         전선 2 (B→A 신호)
    │                              │
    ┤                              ┤
   ═╧═  (접지)              (접지)  ═╧═
  지구가 귀환 경로를 제공
```

**전선 2개**만으로 양방향 통신이 가능하다!

### 5.2 접지의 원리

지구는 거대한 도체이므로 전류의 귀환 경로가 될 수 있다. 물론 지구의 저항은 구리 전선보다 훨씬 높지만, 단면적이 사실상 무한대이므로 전체 저항은 충분히 낮다.

> **Petzold의 비유**: 지구를 흐르는 전류는 마치 넓은 강의 물과 같다. 좁은 수도관(전선)보다 흐름이 느릴 수 있지만, 강이 워낙 넓어서 결국 충분한 양의 물(전류)이 이동한다.

접지 기호는 회로도에서 다음과 같이 표현한다:

```
    │
    ┤
   ═╧═
```

또는

```
    │
   ─┴─
    ─
```

### 5.3 접지의 현대적 의미

현대 전자공학에서 "접지(ground)"는 반드시 지구 땅을 의미하지는 않는다. **기준 전위(reference potential)** — 모든 전압을 측정하는 기준점(0V)을 의미한다.

- 회로의 "-" 또는 공통선 = "접지"
- 배터리의 음극을 접지로 설정하는 것이 일반적

> **핵심 통찰**: 접지의 개념은 전선을 절약하는 실용적 기법에서 시작하여, 현대 전자공학에서는 **기준 전위(0V)**라는 추상적 개념으로 발전했다. 모든 회로에는 접지(기준점)가 있으며, 모든 전압은 이 기준점에 대한 상대값이다.

## 6. 거리의 문제

전선을 길게 설치하면 새로운 문제가 발생한다:

### 6.1 저항 증가

전선은 도체지만 저항이 0이 아니다. 전선이 길어지면 저항이 비례하여 증가한다:

```
짧은 전선 (1m):   저항 작음 → 전류 충분 → 전구 밝음
긴 전선 (100m):   저항 중간 → 전류 감소 → 전구 어두움
매우 긴 전선 (1km): 저항 큼  → 전류 미약 → 전구 거의 안 켜짐
```

옴의 법칙(V = IR)에 의해:
- 전압(V)이 일정할 때, 저항(R)이 커지면 전류(I)가 줄어든다
- 전류가 줄면 전구에 공급되는 에너지가 줄어 빛이 약해진다

### 6.2 해결 시도

| 방법 | 효과 | 한계 |
|------|------|------|
| 더 굵은 전선 사용 | 저항 감소 | 비용 증가, 무거움 |
| 전압 높이기 | 더 센 전류 | 위험, 절연 문제 |
| 중간에 배터리 추가 | 신호 복원 | 누가 중간에서 스위치를 누르나? |

세 번째 방법의 문제를 해결하는 것이 바로 **릴레이(relay)**다. 릴레이는 전기 신호가 자동으로 스위치를 누르게 하는 장치로, Chapter 7에서 상세히 다룬다.

> **핵심 통찰**: 장거리 통신의 핵심 문제는 "신호가 약해진다"는 것이다. 이 문제는 전신 시대의 릴레이에서, 현대의 리피터(repeater), 광섬유 증폭기까지 형태만 달라졌을 뿐 본질은 동일하다 — **약해진 신호를 새로운 에너지로 다시 만들어내기**.

## 7. 이 장에서 확립된 통신 모델

```
[정보 원천]  →  [인코딩]  →  [전송 매체]  →  [디코딩]  →  [수신자]

구체적으로:
[사람의 생각]  →  [모스 부호]  →  [전선의 전류]  →  [모스 부호 해석]  →  [상대방의 이해]
```

이것은 Claude Shannon이 1948년에 정립한 **통신의 수학적 모델**의 기본 구조이기도 하다:

| 구성 요소 | 손전등 (Ch 1) | 전선 (이 챕터) | 현대 인터넷 (Ch 28) |
|-----------|---------------|----------------|---------------------|
| 전송 매체 | 빛 (공기) | 전류 (전선) | 광신호 (광섬유) |
| 장점 | 간단 | 방향 자유 | 초고속, 장거리 |
| 한계 | 직선만 | 저항, 거리 | 인프라 비용 |
| 신호 약화 | 거리² | 거리 비례 | 매우 느림 |

## 회로 구성

양방향 통신 회로의 최종 형태 (접지 사용):

```
[집 A]                              [집 B]

  ┌── 스위치A ──────────────────── 전구B ──┐
  │                                        │
 ─┤├─                                     ─┤├─
 배터리A                                  배터리B
  │                                        │
  ├── 전구A ──────────────────── 스위치B ──┤
  │                                        │
  ┤                                        ┤
 ═╧═                                      ═╧═
 접지                                     접지
```

전선: 2개 (접지 덕분에 귀환선 불필요)

---

## 요약

- 전선을 사용하면 빛의 한계(직선 시야, 날씨)를 극복할 수 있다
- 양방향 통신에는 원래 전선 4개가 필요하지만, **귀환선 공유**로 3개, **접지**로 2개까지 줄일 수 있다
- **접지(ground)**: 지구를 귀환 경로로 사용하는 기법이며, 현대 전자공학에서는 기준 전위(0V)를 의미한다
- 전선이 길어지면 **저항 증가**로 신호가 약해지는 문제가 발생한다
- 이 거리 문제의 해결책이 바로 **릴레이**(Chapter 7)다
