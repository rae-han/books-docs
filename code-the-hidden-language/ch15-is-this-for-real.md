# Chapter 15: Is This for Real? (이것이 진짜인가?)

## 핵심 질문

> 릴레이로 만든 논리 게이트는 실제 컴퓨터에서 사용되는가? 진공관, 트랜지스터, 집적회로는 어떻게 릴레이를 대체했으며, 그 원리는 동일한가?

---

## 1. 릴레이의 한계

지금까지 릴레이로 논리 게이트를 구현했다. 하지만 실제 컴퓨터에 릴레이를 사용하기에는 심각한 문제가 있다:

| 문제 | 설명 |
|------|------|
| **느림** | 기계적 동작 — 접점이 물리적으로 이동해야 함. 초당 수십 회 전환 |
| **큼** | 전자석 + 접점 + 스프링 → 한 개가 손가락 크기 |
| **소음** | 딸깍딸깍 — 수천 개가 동시에 동작하면 공장 소음 |
| **마모** | 기계적 부품이므로 반복 사용 시 닳고 고장남 |
| **소비 전력** | 전자석을 유지하는 데 전력 소모 |

수천 개의 게이트가 필요한 컴퓨터에 릴레이를 사용하면 건물 하나를 채울 크기에, 굉음을 내며, 자주 고장나는 기계가 된다. 실제로 1940년대의 릴레이 컴퓨터(Z3, Harvard Mark I)가 그랬다.

---

## 2. 진공관 (Vacuum Tube)

### 2.1 에디슨 효과

Thomas Edison은 1883년에 전구 실험 중 우연히 발견했다: 가열된 금속에서 전자가 방출된다(**열전자 방출, thermionic emission**). 이것이 "에디슨 효과"다.

### 2.2 다이오드 (Diode)

진공 유리관 안에 가열 필라멘트(음극)와 금속판(양극)을 넣으면:
- 양극이 양전위(+)일 때: 전자가 음극→양극으로 이동 → 전류 흐름
- 양극이 음전위(-)일 때: 전자가 되돌아가지 못함 → 전류 차단

**한 방향으로만 전류를 흐르게 하는 장치** = 다이오드. 교류를 직류로 바꾸는 정류기에 사용.

### 2.3 트라이오드 (Triode)

1906년, Lee de Forest가 다이오드에 **그리드(grid)**라는 제3의 전극을 추가했다:

```
        양극(Plate/Anode)
        ─────────────
        ┃┃┃┃┃┃┃┃┃┃┃ ← 그리드(Grid): 가는 철망
        ─────────────
        음극(Cathode): 가열된 필라멘트
```

그리드에 작은 음전압을 걸면 → 전자 흐름 차단 (스위치 OFF)
그리드에 전압 없음/양전압 → 전자 흐름 허용 (스위치 ON)

**작은 신호(그리드 전압)로 큰 전류(음극→양극)를 제어** = 릴레이와 동일한 원리!

| 비교 | 릴레이 | 진공관 |
|------|--------|--------|
| 제어 수단 | 전자석 (기계적) | 그리드 전압 (전기적) |
| 스위칭 속도 | 수십 Hz | 수백만 Hz |
| 가동 부품 | 있음 (접점) | **없음** |
| 크기 | 크다 | 중간 |

> **핵심 통찰**: 진공관의 혁신은 **기계적 부품 없이** 전기 신호를 제어하는 것이다. 움직이는 부품이 없으므로 릴레이보다 수만 배 빠르고 마모가 없다. 이것이 "전자공학(electronics)"의 시작이다.

### 2.4 진공관 컴퓨터

**ENIAC** (1945): 최초의 범용 전자 컴퓨터
- 약 18,000개의 진공관
- 무게 30톤, 면적 170㎡
- 소비 전력 150kW (집 수십 채 분)
- 초당 수천 회 덧셈 가능

문제: 크기, 발열, 소비 전력, 수명(진공관이 자주 끊어짐)

## 3. 트랜지스터 (Transistor)

### 3.1 반도체 (Semiconductor)

도체(금속)와 절연체(고무) 사이에 **반도체(semiconductor)**가 있다:

| 물질 | 전도성 | 자유 전자 |
|------|--------|-----------|
| 도체 (구리) | 높음 | 매우 많음 |
| **반도체 (실리콘)** | **중간 (제어 가능)** | **조건부** |
| 절연체 (고무) | 거의 없음 | 거의 없음 |

핵심: 반도체의 전도성을 **외부 조건으로 제어**할 수 있다!

### 3.2 도핑 (Doping)

순수 실리콘에 불순물을 소량 첨가하여 전기적 성질을 바꾸는 것:

- **N형 반도체**: 전자가 여분인 불순물(인, 비소) 첨가 → 자유 전자 증가 → **음전하** 운반자
- **P형 반도체**: 전자가 부족한 불순물(붕소, 갈륨) 첨가 → **정공(hole)** 생성 → 양전하 운반자

### 3.3 트랜지스터의 발명

1947년 12월, 벨 연구소의 **John Bardeen**, **Walter Brattain**, **William Shockley**가 트랜지스터를 발명했다. 20세기 가장 중요한 발명 중 하나.

### 3.4 MOSFET — 현대 컴퓨터의 핵심

현대 CPU에서 사용되는 트랜지스터는 **MOSFET(Metal-Oxide-Semiconductor Field-Effect Transistor)**이다:

```
    소스(Source)        드레인(Drain)
        │                   │
   ─────┴───── 채널 ────────┴─────
   │        게이트(Gate)          │
   │            │                 │
              ┌─┴─┐
              │산화│ ← 절연층(SiO₂)
              │막  │
              └───┘
```

- **게이트 전압 없음** → 채널 닫힘 → 소스-드레인 간 전류 차단 (OFF)
- **게이트에 전압 인가** → 채널 열림 → 소스-드레인 간 전류 흐름 (ON)

**작은 전압(게이트)으로 큰 전류(소스-드레인)를 제어** = 릴레이, 진공관과 동일한 원리!

| 비교 | 릴레이 | 진공관 | 트랜지스터 |
|------|--------|--------|-----------|
| 제어 | 전자석 | 그리드 전압 | 게이트 전압 |
| 속도 | ~100 Hz | ~MHz | **~GHz** |
| 크기 | 수 cm | 수 cm | **수 nm** |
| 소비 전력 | 높음 | 높음 | **극히 낮음** |
| 수명 | 유한 (마모) | 수천 시간 | **사실상 무한** |

> **Petzold의 비유**: 릴레이, 진공관, 트랜지스터는 모두 "전기로 제어하는 스위치"라는 같은 역할을 한다. 마치 양초, 가스등, 전구가 모두 "빛을 내는 것"이라는 같은 역할을 하지만 기술이 다른 것처럼. **논리 게이트의 기능은 구현 기술이 바뀌어도 동일하다.**

## 4. 집적회로 (Integrated Circuit, IC)

### 4.1 발명

- **1958년**: Jack Kilby (Texas Instruments) — 최초의 집적회로
- **1959년**: Robert Noyce (Fairchild) — 실리콘 기반 평면 공정

**아이디어**: 트랜지스터, 저항, 커패시터를 **하나의 실리콘 칩**에 모두 제작.

### 4.2 집적도의 발전

| 약어 | 이름 | 트랜지스터 수 | 시기 |
|------|------|-------------|------|
| SSI | Small-Scale | ~10개 | 1960s |
| MSI | Medium-Scale | ~100개 | 1960s-70s |
| LSI | Large-Scale | ~1,000개 | 1970s |
| VLSI | Very Large-Scale | ~10,000+ | 1980s |
| ULSI | Ultra Large-Scale | 수백만~수십억 | 1990s~ |

현대 CPU의 트랜지스터 수:

| 프로세서 | 연도 | 트랜지스터 수 |
|----------|------|-------------|
| Intel 4004 | 1971 | 2,300 |
| Intel 8080 | 1974 | 6,000 |
| Intel 386 | 1985 | 275,000 |
| Pentium | 1993 | 3,100,000 |
| Apple M1 | 2020 | 16,000,000,000 |

### 4.3 무어의 법칙 (Moore's Law)

Intel 공동 창업자 **Gordon Moore**가 1965년에 관찰: **집적회로의 트랜지스터 수가 약 2년마다 2배로 증가한다.**

이것은 물리 법칙이 아니라 경험적 관찰/산업 목표이지만, 50년 이상 놀라울 정도로 정확하게 유지되었다. 최근에야 물리적 한계에 다가가면서 둔화되고 있다.

> **핵심 통찰**: 릴레이 → 진공관 → 트랜지스터 → IC의 발전에서 변하지 않는 것은 **논리 게이트의 기능**이다. AND는 항상 AND이고, NOT은 항상 NOT이다. 변한 것은 크기, 속도, 비용뿐이다. 이것이 **추상화(abstraction)**의 힘이다 — Chapter 8에서 게이트를 "블랙박스"로 취급할 수 있었던 이유가 바로 이것이다.

## 요약

- **진공관**: 가열된 음극에서 전자를 방출, 그리드 전압으로 전류 제어. 릴레이보다 수만 배 빠르지만 크고 뜨겁다
- **트랜지스터**: 반도체(실리콘)에 도핑하여 만든 전자 스위치. 게이트 전압으로 전류 제어. 작고, 빠르고, 안정적
- **집적회로(IC)**: 하나의 실리콘 칩에 수십억 개의 트랜지스터를 집적
- **무어의 법칙**: 약 2년마다 트랜지스터 수 2배 — 50년 이상 유지
- **핵심 원리는 동일**: 릴레이든 트랜지스터든 "작은 신호로 큰 신호를 제어"하여 논리 게이트를 구현하는 것. 추상화 덕분에 구현 기술이 바뀌어도 논리 설계는 그대로 유효하다
