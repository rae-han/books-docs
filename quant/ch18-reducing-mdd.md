# Chapter 18: MDD를 줄이는 방법

## 핵심 질문

연복리 40~50%를 내는 전략들이 *MDD 50~60%*라면 실전에서 살아남을 수 있는가? 워런 버핏의 첫 번째 원칙 *"돈을 잃지 말라"*를 어떻게 지킬 것인가? *손익 비대칭성*이란 무엇이고 왜 MDD를 *20% 이하*로 유지해야 하는가? 그리고 자산배분·마켓타이밍·거래량 필터로 MDD를 어떻게 줄일까?

이 챕터는 책의 가장 중요한 챕터 중 하나로, *손실 최소화*를 통해 강력한 전략을 *실전 가능한 전략*으로 바꾸는 방법을 다룬다.

## 1. 투자에서 가장 중요한 원칙 — 손실 최소화

### 워런 버핏의 두 원칙

> **첫 번째 원칙**: 돈을 잃지 말라 (Don't lose money)
> **두 번째 원칙**: 첫 번째 원칙을 잊지 말라 (Don't forget rule no. 1)

초보 투자자의 원칙(_"수익을 극대화하자!"_)과 정반대. 왜?

### 손익 비대칭성의 원리

```
퀴즈: 50% 손실이 발생했을 경우, 원금을 회복하려면 몇 % 수익이 필요할까?

❌ 오답: 50%
✅ 정답: 100% (1만 원 → 5,000원이 다시 1만 원이 되려면 2배 필요)
```

| 손실률  | 본전 회복에 필요한 수익률 |
| ------- | ------------------------- |
| 10%     | 11%                       |
| 20%     | 25%                       |
| 25%     | 33%                       |
| 33%     | 50%                       |
| **50%** | **100%**                  |
| 66.7%   | 200%                      |
| 75%     | 300%                      |
| 90%     | 900%                      |
| 95%     | 1,900%                    |

> **핵심 통찰**: 손실 규모가 커질수록 *본전 만회 이익*이 *기하급수적*으로 커진다. 90% 잃으면 900% 수익을 내야 본전 — _동화의 영역_.

### MDD 20% 절대 한도

> 의미 있는 금액을 투자하고 포트폴리오가 20% 이상 깨지면, 즉 손실이 20%를 넘어서면 _편하게 잠을 잘 수 있는 사람이 거의 없다_. 일상 생활이 손에 안 잡히고 우울증, 불면증, 극심한 스트레스를 호소한다.

### 강환국의 가슴 아픈 투자 사례 (영상 197)

> 2010년 부채까지 끌어와 서울 아파트를 매수. 2013년까지 가격이 쭉쭉 빠졌다. 손실이 순자산의 20~30%. 2014년 본전 회복 후 _"이제 팔아야지!"_ 하고 팔았다. 2014~2021년 서울 아파트 폭등을 _고스란히 놓쳤다_. 마음고생이 너무 심해서.

> **핵심 통찰**: 손실회피 편향과 처분효과의 교과서적 사례. 이 경험이 *비자발적 장기투자*의 비극을 만든다.

### 비자발적 장기투자의 두 시나리오

```
시나리오 1 (그나마 운 좋음):
  몇 년 걸려 본전 회복 → 잽싸게 매도 (수익률 0%)
  추가 상승은 놓침
  자산군 자체에 진절머리 → 다시는 안 봄

시나리오 2 (최악):
  영영 회복 안 됨 / 상장폐지
  자산의 50%, 70%, 100% 손실 → 재기 불가능
```

> **핵심 통찰**: **MDD 20% 이상인 전략으로는 절대 투자하지 말자!**

### 14~16장 전략의 큰 문제

> 자, 여기서 큰 문제가 발생한다. 14~16장에서 소개한 전략들은 기대수익률은 매우 높지만, *개별 종목에 1년 내내 투자하는 전략의 MDD는 50~60%대*다.

5~10년에 한 번 주가지수가 반토막 나는 시기가 오고, 그때는 모든 주식이 폭락한다. 따라서 어떤 분산도 이 정도 MDD는 못 피한다.

> **핵심 통찰**: 우리의 과제 — _어떻게 초과수익을 일부라도 유지하면서 MDD를 20% 이하로 유지할 것인가?_

## 2. PRACTICE — 자산배분 + 주식 비중만 퀀트 (영상 505)

### 영구 포트폴리오 (Permanent Portfolio)

| 분류     | 매수 전략                                                                                     |
| -------- | --------------------------------------------------------------------------------------------- |
| **분류** | 자산배분                                                                                      |
| **매수** | 미국 주식(SPY), 미국 장기국채(TLT), 금(GLD), 미국 초단기 채권(BIL) — 4개 ETF에 _자산을 4등분_ |
| **매도** | 연 1회 리밸런싱                                                                               |

### 영구 포트폴리오 백테스트 (1970/01~2021/05, 50년)

| 항목              | 결괏값    |
| ----------------- | --------- |
| 초기 자산         | $10,000   |
| 최종 자산         | $733,440  |
| **연복리 수익률** | **8.7%**  |
| 표준편차          | 7.2%      |
| 수익난 월 비율    | 65%       |
| **MDD**           | **12.7%** |
| 샤프 지수         | 0.55      |

> **핵심 통찰**: 연복리 8.7%는 14~16장 전략보다 낮지만 _MDD 12.7%_ — 50년간 최악의 순간에도 12.7%만 잃었다.

### 비결: 상관성 0인 자산 결합

| 자산      | 단독 MDD |
| --------- | -------- |
| 미국 주식 | 50.89%   |
| 미국 채권 | 23.12%   |
| 금        | 61.78%   |

| 자산     | 미국주식 | 미국채권 | 금    | 달러현금 |
| -------- | -------- | -------- | ----- | -------- |
| 미국주식 | —        | 0.02     | 0.05  | 0.01     |
| 미국채권 | 0.02     | —        | 0.07  | 0.04     |
| 금       | 0.05     | 0.07     | —     | -0.03    |
| 달러현금 | 0.01     | 0.04     | -0.03 | —        |

> **핵심 통찰**: 4개 자산의 상관성이 _거의 0_ → 한 자산이 망해도 다른 자산이 같이 떨어지지 않음. 게다가 4개 모두 _장기 우상향_. 자산 하나가 하락해도 나머지 3개의 상승폭이 더 크다.

### 주식 비중을 퀀트 전략으로 대체

> 그러면 주식 비중을 *밸류 + 모멘텀 전략*으로 바꾸면 어떨까?

| 분류     | 매수 전략                                                                                 |
| -------- | ----------------------------------------------------------------------------------------- |
| **분류** | 자산배분 + 퀀트                                                                           |
| **매수** | 한국 주식(밸류+모멘텀 20개), 미국 장기국채(TLT), 금(GLD), 미국 초단기 채권(BIL) — _4등분_ |
| **매도** | 4월·10월 마지막 거래일 (연 2회)                                                           |

### 백테스트 (2007/01~2021/05)

| 포트폴리오                       | 연복리     | 표준편차 | MDD       | 샤프     |
| -------------------------------- | ---------- | -------- | --------- | -------- |
| 영구 포트폴리오 (오리지널)       | 6.77%      | 6.77     | 13.7%     | 0.86     |
| **영구 + 밸류+모멘텀 한국 주식** | **15.64%** | 9.18     | **16.2%** | **1.55** |

> **핵심 통찰**: 주식 비중만 퀀트로 바꾸면 _수익률 압도적 상승_, MDD는 약간 증가하지만 _여전히 20% 이하_. 샤프 지수 1.55는 매우 우수.

> "퀀트 전략은 연복리 44%를 벌었는데 겨우 15%에 만족하라고?" — 답은 *MDD 16% 이하 유지*가 그만한 가치가 있다는 것.

## 3. PRACTICE — 마켓타이밍 (영상 505)

### 부지런한 투자자의 길

> 지금까지 소개한 전략은 대부분 _연 1~2회 리밸런싱_. 더 부지런한 투자자는 *마켓타이밍*으로 MDD를 획기적으로 줄일 수 있다.

### 『현명한 퀀트주식투자』의 코스닥 이동평균 마켓타이밍

| 룰                                                                |
| ----------------------------------------------------------------- |
| **A**: 코스닥 종가 - 코스닥 종가 3일 이동평균 > 0                 |
| **B**: 코스닥 종가 - 코스닥 종가 5일 이동평균 > 0                 |
| **C**: 코스닥 종가 - 코스닥 종가 10일 이동평균 > 0                |
| → A or B or C 일 때만 주식 매수, 셋 다 음수면 _다음 날 전량 매도_ |

### 적용 사례 (2021년 2월)

```
2/8: 종가 960.78 < 3일/5일/10일 이동평균 → 다음 날 매도
2/10: 종가 964.31 > 모든 이동평균 → 다음 거래일(2/15) 매수
2/18: 종가 967.42 < 모든 이동평균 → 2/19 매도
...
```

평균 보유 기간 _3~4일_. 매우 잦은 거래.

### 마켓타이밍 적용 vs 미적용 (2007/01~2020/04)

| 전략명                     | 마켓타이밍 미적용 MDD | 마켓타이밍 적용 MDD |
| -------------------------- | --------------------- | ------------------- |
| 조엘 그린블라트 마법공식   | 66.0%                 | **13.5%**           |
| 조셉 피오트로스키 F-스코어 | 55.8%                 | **7.9%**            |
| 벤저민 그레이엄 NCAV       | 54.0%                 | **15.4%**           |
| 피터 린치 PEG              | 60.8%                 | 16.8%               |
| 데이비드 드레먼 역발상     | 65.0%                 | 15.2%               |
| 월터 슐로스                | 45.1%                 | 13.2%               |
| 존 네프 GYP 비율           | 39.3%                 | 18.4%               |
| 켄 피셔 PSR 전략           | 49.0%                 | 16.8%               |

> **핵심 통찰**: 모든 전략의 MDD가 _20% 이하로 감소_. 최소 53%(존 네프) ~ 최대 85%(피오트로스키) 감소. 게다가 백테스트 연복리 _30~50%_ 유지.

### 마켓타이밍의 한계

| 한계                                                              |
| ----------------------------------------------------------------- |
| 평균 보유 기간 3~4일 — 일반 투자자가 _수동으로 불가능_            |
| _자동매매 시스템 (뉴지스탁 젠트레이더)_ 필수                      |
| 전략별 투자 가능 금액이 _최대 2천만 원 정도_ (저자가 책에서 인용) |

> **핵심 통찰**: 저자도 2021년 말부터 젠트레이더를 활용해 마켓타이밍 적용 계획. *수동 트레이딩은 불가능*하므로 자동매매 도구가 필수.

## 4. PRACTICE — 거래량과 슬리피지 영향 (영상 441)

### 우려와 검증

소형주 전략에서 거래량 적은 종목 때문에 *슬리피지(slippage)*가 클 가능성. 저자가 직접 백테스트:

### 밸류 + 모멘텀 소형주 전략 (2007/06~2021/05)

| 슬리피지 (%) | 5일 평균 최소 거래대금 | 연복리 (%) | MDD (%) |
| ------------ | ---------------------- | ---------- | ------- |
| 0%           | 없음                   | 56.6       | 50.3    |
| 0.5%         | 없음                   | 55.3       | 50.8    |
| 1%           | 없음                   | 54.0       | 51.3    |
| 0.5%         | 1억                    | 55.1       | 55.7    |
| 1%           | 1억                    | 50.9       | 57.8    |
| 1%           | 2억                    | 51.1       | 56.1    |
| 2%           | 2억                    | 40.9       | 53.1    |

| 결론                                                         |
| ------------------------------------------------------------ |
| **슬리피지** — 거래 빈도 낮음(연 2회)이라 큰 영향 없음       |
| **5일 평균 거래대금 1억 이하 제거** — 수익률 거의 영향 없음  |
| **5일 평균 거래대금 2억 이하 제거** — 수익률 _급격히 떨어짐_ |

> **핵심 통찰**: 평균 거래대금 5천만~1억 원인 주식에 *1,000만 원 정도 투자*는 별로 어렵지 않다. 1억 원 필터는 안전, 2억은 너무 강함.

### 저자의 실전 경험

> 오전 9~10시에 최근 체결가격 근처에 *지정가 매수*를 걸어놓고 HTS를 끄면 그날 종가까지 매수에 성공. 100개 종목 중 거래량 없는 3~4개만 다른 종목으로 대체.

> **핵심 통찰**: 이 책에서 수익률이 매우 높았던 14년·20년 백테스트 결과는 대부분 *이런 생소한, 투자자의 관심을 전혀 받지 못하는 기업*을 사고 팔아서 이뤄진 것이다.

| 핫한 기업                         | 생소한 기업                  |

> 결론: _"눈 딱 감고 사용하는 전략에 맞는 계량 지표를 지닌 주식들을 그냥 사세요!"_

## 6. 18장 전략 종합

### MDD 줄이기 옵션

| 방법                           | 수익률 영향    | MDD 효과         | 난이도          |
| ------------------------------ | -------------- | ---------------- | --------------- |
| **11~4월만 투자**              | 약간 감소      | 절반 가까이 감소 | 낮음            |
| **자산배분 + 주식만 퀀트**     | 큰 폭 감소     | 16% 수준 유지    | 낮음            |
| **마켓타이밍 (3·5·10일 이평)** | 유지 또는 증가 | 20% 이하로 급감  | _자동매매 필수_ |
| **거래량 필터**                | 약간 감소      | 약간 영향        | 낮음            |

저자의 실전 조합:

> 나는 *11~4월 투자*와 _자산배분의 주식 비중을 퀀트 전략으로_ 방법을 사용하고 있는데, 이렇게 MDD를 획기적으로 줄일 수 있는 방법을 알고 깜짝 놀랐다. 2021년 말부터 *젠트레이더를 활용해 마켓타이밍 전략*도 적용할 계획이다.

## 7. 18장의 핵심 통찰

> **핵심 통찰 1**: _손익 비대칭성_ — 50% 잃으면 100% 벌어야 본전. 손실이 클수록 *기하급수적*으로 회복이 어려워진다.

> **핵심 통찰 2**: 의미 있는 금액을 투자할 때 _MDD 20% 초과_ 시 일상이 흔들린다. 비자발적 장기투자의 함정에 빠진다.

> **핵심 통찰 3**: 14~16장 모든 전략의 MDD는 50% 이상. 그대로 사용하면 _실전 생존 불가능_. MDD를 20% 이하로 줄이는 것이 18장의 과제.

> **핵심 통찰 4**: _영구 포트폴리오 + 주식 비중만 퀀트_ — 4등분으로 MDD 16%, 연복리 15.6%. 가장 단순하고 효과적.

> **핵심 통찰 5**: _마켓타이밍 (코스닥 3·5·10일 이평)_ — 모든 전략의 MDD를 20% 이하로 떨어뜨림. 단, _자동매매 필수_ (수동 불가능).

> **핵심 통찰 6**: 거래대금 5일 평균 *1억 원 필터*는 안전. 2억은 너무 강함.

> **핵심 통찰 7**: 퀀트 전략의 종목은 *생소한 기업*들. 그게 정상 — 무시당하는 기업이 *저평가의 원천*이기 때문.

## 요약

- 손익 비대칭성: 50% 손실 → 100% 수익이 본전. 손실은 *기하급수적*으로 회복이 어려움.
- _MDD 20%_ 절대 한도. 그 이상이면 비자발적 장기투자 → 시나리오 1(0% 수익) 또는 시나리오 2(파산).
- 14~16장 전략의 MDD 50~60% — 실전엔 부족. _18장이 해결_.
- **자산배분 + 주식만 퀀트**: 영구 포트폴리오 4등분 중 주식만 한국 퀀트. 연복리 15.6%, MDD 16.2%.
- **마켓타이밍**: 코스닥 3·5·10일 이동평균 — 모든 전략 MDD 20% 이하로. 자동매매 필수.
- **거래량 필터**: 5일 평균 거래대금 1억 원 이하 제거가 적절. 2억은 너무 강함.
- **종목 선택**: 퀀트가 추천하면 _생소해도 그냥 사라_. 거래량 부족·산업 쏠림만 예외.

## 다른 챕터와의 관계

이 챕터는 **책 전체에서 가장 중요한 챕터** 중 하나다. 14~16장의 강력한 전략들이 _18장 없이_ 실전에서 살아남을 수 없다.

17장(전략 선택)이 "*무엇*을 살까"였다면 18장은 "_어떻게_ 위험을 피할까". 두 챕터를 합쳐야 비로소 *실전 가능한 투자 시스템*이 완성된다.

다음 *19장*은 _메타 분석_ — "이 모든 전략이 미래에도 통할까?", "과최적화는 아닐까?", "왜 장기적으로 투자자가 돈을 못 버는가?"라는 더 큰 질문을 다룬다.

본 챕터의 *마켓타이밍*은 『현명한 퀀트주식투자』(2021, 닥터퀀트 외)에서 더 깊이 다뤄지므로, 자동매매를 진지하게 고민한다면 그 책 참고 권장.
