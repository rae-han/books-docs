# Chapter 16: 모멘텀 + 울트라 전략

## 핵심 질문

밸류와 퀄리티에 이어 *모멘텀 팩터*는 어떻게 활용할 것인가? 영업이익·순이익이 성장하는 *이익 모멘텀*은 한국 시장에서 얼마나 강력할까? 밸류 + 모멘텀, 퀄리티 + 모멘텀의 조합 시너지는? 그리고 모든 팩터를 합친 **강환국 울트라 전략**의 백테스트 결과는?

이 챕터는 *Part 4(새로운 투자 전략)*의 클라이맥스로, 14~15장의 누적된 팩터를 *최종 통합*한다.

> **핵심 통찰**: 우량주가 왜 수익률이 높은가? 가치주(밸류)와 모멘텀 주식의 수익률이 높은 이유와 마찬가지로, *투자자의 심리적 편향*이 핵심이다.

## 1. 우량주가 수익률이 높은 3가지 이유

### ① 저변동 주식의 우위

> 짧은 시간에 엄청난 돈을 벌려면 어떤 주식을 사야 할까? 핫한 _변동성이 어마어마한_ 주식이나 암호화폐다. 운이 좋으면 한 달에 몇 배 — _복권 같은_ 수익. 반대로 변동성이 적은 주식은 짧은 시간에 큰 수익이 어렵다.

| 결과                                                                                   |
| -------------------------------------------------------------------------------------- |
| "복권 식 투자"를 원하는 이들이 *변동성 높은 주식*의 가격을 필요 이상으로 올림 (고평가) |
| 재미없는 *변동성 낮은 주식*은 평가절하 (저평가)                                        |
| 시간이 지나면 _저변동 주식이 평균적으로 더 높은 수익_ — 투자의 정석                    |

### ② 재무건전성 높은 기업의 우위

부채·차입금 적은 무난한 기업은 _재미가 없다_. 망할 가능성도 없지만 짧은 시간 큰 수익도 어렵다. 반대로 부채 많고 파산 가능성 높은 기업은 *컴백에 성공하면 수십 배 수익*도 가능 — 투자자들이 과잉확신으로 이런 기업을 선호.

```
복권 심리:    부채 많은 기업 → 고평가
시장 외면:    부채 적은 기업 → 저평가
시간 후:      재무건전성 높은 기업이 평균적으로 더 높은 수익
```

### ③ 고수익성 기업의 우위

워런 버핏: "좋은 기업은 *자본 대비 수익성이 높은 기업*이다."

> 그런데 투자자들은 의외로 수익성에 관심이 없다. 대부분은 _최근 편향_ 때문에 시야가 좁아 시황·소문·테마·뉴스에 더 집착한다. 핫이슈가 없는 기업은 우수한 수익성이 가격에 반영되지 않아 저평가된다.

## 2. 이익 모멘텀의 발견

저자가 백테스트로 알아낸 가장 중요한 사실:

> **핵심 통찰**: *분기 영업이익 성장*과 *분기 순이익 성장*이 한국 주식 시장에서 매우 막강한 팩터다. 이를 *이익 모멘텀*이라 명명한다.

| 특징                                                                            |
| ------------------------------------------------------------------------------- |
| 11~4월에서도 잘 먹힌다                                                          |
| 소형주에도 잘 먹힌다                                                            |
| **밸류 지표와 시너지 매우 높다** — 저평가 + 이익 급성장 기업의 수익이 매우 높다 |

## 3. 이익 모멘텀 + 소형주 + 계절성 (영상 458)

### 이익 모멘텀 전략 룰

| 분류          | 매수 전략                                                               |
| ------------- | ----------------------------------------------------------------------- |
| **분류**      | 모멘텀                                                                  |
| **매수**      | 시가총액 하위 20% 기업에서 4개 지표 평균 순위 상위 매수 (모두 내림차순) |
|               | • 전 분기 대비 영업이익 성장률                                          |
|               | • 전년 동기 대비 영업이익 성장률                                        |
|               | • 전 분기 대비 순이익 성장률                                            |
|               | • 전년 동기 대비 순이익 성장률                                          |
| **매수 종목** | 20개                                                                    |
| **매도**      | 4월·10월 마지막 거래일 (연 2회 리밸런싱)                                |

### 이익 모멘텀 소형주 전략 (2007/06~2021/05)

| 항목           | 결괏값    |
| -------------- | --------- |
| 누적 수익률    | 29,778.7% |
| 연 평균 수익률 | **50.2%** |
| MDD            | 56.6%     |
| 승률           | 63.7%     |

> **핵심 통찰**: 연복리 50% — 14년간 원금이 *298배*가 된다. 단, MDD 56.6%로 위험도 매우 높다.

### 이익 모멘텀 소형주 + 11~4월

| 항목           | 결괏값    |
| -------------- | --------- |
| 누적 수익률    | 4,500.7%  |
| 연 평균 수익률 | **31.2%** |
| MDD            | **21.7%** |
| 승률           | 73.8%     |

> **핵심 통찰**: 연복리는 50% → 31%로 줄지만 MDD가 56% → 22%로 _대폭 감소_. 이쪽이 실전에 더 적합.

## 4. 밸류 + 모멘텀 전략 (영상 504)

### 두 전략의 시너지 가설

> 이쯤 읽으면 이익 모멘텀에 *밸류 지표*를 붙이면 시너지가 날지 궁금할 만하다. 이익이 크게 성장하는데 아직 저평가된 기업을 매수하는 것이다.

### 매수 전략

| 분류           | 매수 전략                                                                                          |
| -------------- | -------------------------------------------------------------------------------------------------- |
| **분류**       | 밸류 + 모멘텀                                                                                      |
| **모멘텀 4개** | 전 분기 영업이익 성장률, 전년 동기 영업이익 성장률, 전 분기 순이익 성장률, 전년 동기 순이익 성장률 |
| **밸류 4개**   | 분기 1/PER, 분기 1/PFCR, 1/PBR, 분기 1/PSR                                                         |
| **매수**       | 8개 평균 순위 상위 매수                                                                            |
| **매수 종목**  | 20개                                                                                               |
| **매도**       | 4월·10월 (연 2회)                                                                                  |

### 백테스트 (2007/06~2021/05)

| 전략                       | 연복리    | MDD   |
| -------------------------- | --------- | ----- |
| 이익 모멘텀                | 28.1%     | 57.2% |
| 강환국 슈퍼가치 (PFCR)     | 38.3%     | 43.8% |
| **밸류 + 모멘텀**          | **44.0%** | 45.0% |

> **핵심 통찰**: 28.1% + 38.3% = 44% (시너지 발생). 소형주에서는 *56.6%*로 폭발.

### 11~4월의 한계

> 처음으로 11~4월 전략에서 *만족스럽지 않은 성과*가 나왔다. 1년 투자: 연복리 44% / MDD 45%, 11~4월 투자: 연복리 22.6% / MDD 31%. _MDD가 낮아진 것에 비해 연복리가 너무 줄었다_.

## 7. 대형주 울트라 전략

### 왜 대형주도 필요한가

> 한국 시장에서 소형주가 2000~2020년 전체 구간에 대형주를 압도한 것은 사실이지만, 반대로 *수년간 대형주가 더 높은 시기*도 있다. 2017년처럼 주가지수와 몇몇 대형주는 잘 나갔지만 소형 가치주는 부진한 시기. 이때 대형주를 보유하지 않으면 *큰 소외감*을 느낀다.

### 매수 전략

| 분류          | 매수 전략                                                   |
| ------------- | ----------------------------------------------------------- |
| **분류**      | 밸류 + 퀄리티 + 모멘텀 (대형주)                             |
| **모집단**    | 시가총액 _상위 20%_ (대형주)                                |
| **나머지**    | 강환국 울트라 전략과 동일 (12개 평균 순위, 신 F-스코어 3점) |
| **매수 종목** | 20개                                                        |

### 백테스트

| 백테스트 결과                                         |
| ----------------------------------------------------- |
| 연복리 수익률 _20% 이상_ (정확한 수치는 책 본문 참고) |
| 상위 10% 주식은 5~10월 수익률도 양호                  |
| KODEX 200 ETF 대비 우월                               |

> **핵심 통찰**: 단순 KODEX 200 ETF에 투자하기보다 *대형주 울트라*가 더 높은 수익. 대형주 비중이 필요하다면 이 전략 추천.

## 9. 16장의 핵심 통찰

> **핵심 통찰 1**: *이익 모멘텀*은 한국 시장에서 가장 강력한 팩터 중 하나. 소형주 + 이익 모멘텀 = 연복리 50%.

> **핵심 통찰 2**: 밸류와 모멘텀의 결합이 가장 강력 — _밸류+모멘텀 소형주 = 연복리 56.6%_. 14년간 원금 535배.

> **핵심 통찰 3**: 강환국 울트라(밸류+퀄리티+모멘텀 12개) 소형주는 *연복리 42.7%*로 19년 반에 1,231배. 5~10월에도 14% 수익 — _계절성 덜 의존_.

> **핵심 통찰 4**: 십분위 분포가 완벽한 단조 관계 — *팩터 신뢰성*이 입증됨.

> **핵심 통찰 5**: 대형주 울트라도 KODEX 200 ETF 대비 우월. *대형주 비중*이 필요한 이유는 소외감 회피.

## 요약

- 우량주가 수익률 높은 3가지 이유: _저변동 우위_, _재무건전성 우위_, _고수익성 우위_. 모두 투자자 심리 편향에서 비롯.
- **이익 모멘텀** = 분기 영업이익·순이익 성장률 4개. 소형주 연복리 50.2% (MDD 56.6%).
- **밸류 + 모멘텀** = 8개 팩터. 소형주 연복리 56.6%. 11~4월만 투자는 만족스럽지 않음 (예외).
- **퀄리티 + 모멘텀** = 8개 팩터 + 신 F-스코어 3점 필터. 소형주 36.1%.
- **강환국 울트라** = 밸류·퀄리티·모멘텀 12개 + 신 F-스코어 3점. 전체 34.15%, 소형주 42.73%. 19년 반에 1,231배.
- 십분위 분포가 _완벽한 단조_ — 팩터 신뢰성 입증.
- *대형주 울트라*도 연복리 20%+ — 대형주 비중 필요할 때 추천.
- 모든 전략의 매도: 4월·10월 마지막 거래일 연 2회.

## 다른 챕터와의 관계

이 챕터는 *Part 4(새로운 투자 전략)*의 **클라이맥스**다. 14장(밸류) → 15장(퀄리티 + 밸류·퀄리티 통합) → 16장(모멘텀 + 모든 통합)의 점진적 빌드업이 *강환국 울트라*에서 정점에 달한다.

다음 *17장*은 "이렇게 많은 전략 중 어떤 것을 골라야 하는가?"라는 _전략 선택_ 문제를 다룬다. 16장의 울트라 전략이 가장 강력해 보이지만, 17장은 *취향대로*와 *분산 투자*가 더 현명할 수 있다고 주장한다.

*18장(MDD를 줄이는 방법)*은 16장의 강력한 전략들이 가진 *MDD 50% 이상*이라는 치명적 약점을 극복하는 방법을 제시한다.

16장의 유튜브 채널 *『할 수 있다! 알고투자』*에서 영상 503, 504, 520, 521 등을 직접 시청하면 더 깊이 이해할 수 있다.
