# Chapter 17: 어떤 전략에 투자해야 하는가?

## 핵심 질문

수십 개 전략을 백테스트했는데 어떤 것을 실제로 골라야 하는가? *그린블라트의 마법공식*처럼 한 개 전략을 강력 추천할 수 있는가, 아니면 다수 전략에 분산해야 하는가? 그리고 *팩터 모멘텀*이라는 트렌디한 방법은 무엇인가?

> 그레잇! 그런데 오히려 전략이 너무 많아서 혼란에 빠질 수 있다. _"아니, 왜 그린블라트의 『주식시장을 이기는 작은 책』처럼 한 개 전략을 제공하는 게 아니라 헷갈리게 이렇게 많아?"_ 라고 한탄하는 독자도 있을 것이다.

이 챕터는 14~16장에서 만든 수많은 전략 *중에서 무엇을 고를지*를 결정하는 가이드다.

## 1. 세 가지 선택 방법

저자가 제안하는 정답에 근접한 방법:

| 방법                              | 핵심               |
| --------------------------------- | ------------------ |
| **1. 자기 취향에 맞는 전략 선택** | 가장 심플          |
| **2. 전략 분산 투자**             | 위험 분산 효과     |
| **3. 팩터 모멘텀 따르기**         | 최근 트렌디한 방법 |

> **핵심 통찰**: 백테스트 수익은 *과거 수익*일 뿐이다. 미래에 얼마든지 뒤집힐 수 있다. 백테스트 자체로 결정하기보다는 _"시장보다 많이 좋다 / 좀 좋다 / 비슷하다 / 별로다 / 매우 안 좋다"_ 5단계로 평가만 하라. *시장보다 많이 좋다*는 평가가 섰으면 취향대로 투자해도 좋다.

## 2. PRACTICE — 취향대로 투자하기

### 투자자 100명, 취향 100개

> 투자자가 100명이라면 취향이 100개 있을 거라고 생각해서 일부러 여러 전략을 제공했다.

| 투자자 유형        | 선호 전략                   |
| ------------------ | --------------------------- |
| 그레이엄 스타일    | 초저평가 (NCAV)             |
| 버핏 스타일        | 우량 가치주                 |
| 피터 린치 스타일   | 성장주                      |
| 조지 소로스 스타일 | 모멘텀 주식                 |
| 배당주 선호        | 꼬박꼬박 들어오는 배당 매력 |

> **핵심 통찰**: 초저평가된 주식을 선호하는 사람은 성장주에 투자하면 안 된다. 백테스트 수익률이 아무리 화려하더라도, *저 PBR과 NCAV를 따지던 사람*이 영업이익 성장률 좀 높다고 *PER 50, PBR 6 기업*에 투자할 수 있을까? 또는 대형주 선호 투자자가 갑자기 *듣도 보도 못한 소형주에 전 재산 베팅*할 수 있을까?

심리적으로 매우 힘들다. 그래서 자신의 취향이 가장 중요하다.

### 저자의 취향 변화

저자의 자기 고백:

| 시기               | 선호 전략                                       |
| ------------------ | ----------------------------------------------- |
| 2006년 (퀀트 시작) | 버핏 영향: _우량 가치주_ (저 PBR + 고 F-스코어) |
| 그 후              | 그레이엄의 _초저평가주_ (NCAV)도 매력           |
| 한때               | 두 전략에 자산의 50%씩                          |
| 요즘               | 선호가 많이 없어져서 밸류·퀄리티·모멘텀 통합    |

## 3. PRACTICE — 전략 분산 투자

### 분산 투자의 의미

> 자신의 취향대로 투자하는 것이 좋다고 했지만, 나처럼 취향이 없는 경우는 전략 내에서 최대한 분산투자하는 것을 추천한다.

여기서 분산이란 *수십 개 전략에 다 투자하라*는 게 아니라:

```
밸류 / 퀄리티 / 모멘텀
대형주 / 소형주
                  ← 골고루 분산
```

### 시장 성격에 따른 팩터 우위

| 시기          | 우위                                   |
| ------------- | -------------------------------------- |
| 1990년대      | 성장·모멘텀 강세 (기술주). 밸류 부진   |
| 2000년대      | 모멘텀 버블 터지면서 밸류 회복         |
| 2010년대 후반 | 가치주 극도로 소외, 기술주 모멘텀 강세 |
| 2017년        | 대장주·주가지수 강세, 소형주 소외      |

> **핵심 통찰**: 미래에 어떤 테마·팩터·시가총액이 잘 나갈지 _절대 예측 불가능_. 그래서 모든 팩터·시가총액에 분산.

### 분산 투자 사례

```
❌ 분산 안 됨: NCAV + 강환국 슈퍼가치  (둘 다 밸류)

✅ 잘 분산됨: 강환국 슈퍼가치 + 슈퍼퀄리티 + 이익 모멘텀
   → 밸류·퀄리티·모멘텀 모두 커버

✅ 더 잘 분산: 밸류+모멘텀 대형주 + 슈퍼퀄리티 소형주 + 밸류+모멘텀 소형주
   → 팩터·시가총액 모두 커버 (소형주 비중 늘림)
```

### 한 전략당 종목 수

> 여러 전략을 쓸 때 한 전략에 매수하는 종목 수는 *20개*가 적합하다고 생각한다. 그래서 퀀트킹으로는 늘 '20개 종목 포트폴리오'를 백테스트한 것이다.

### 분산 투자의 한계

> **핵심 통찰**: 여러 팩터·시가총액에 분산해도 _MDD를 획기적으로 낮출 수는 없다_. 주식 자산군은 급락장에서 _상관성이 높아져_ 다 같이 나락으로 간다. 주가지수가 반토막 나면 모든 전략이 최소 반토막 난다.

이 한계를 극복하는 방법이 다음 *18장(MDD 줄이는 방법)*의 주제다.

## 4. PRACTICE — 팩터 모멘텀

| 개념            | 설명                                               |
| --------------- | -------------------------------------------------- |
| **팩터 모멘텀** | 최근 잘 나간 팩터가 가까운 미래에도 잘 나갈 가능성 |
| **활용**        | 최근 1~3년 수익률 상위 팩터에 비중                 |
| **위험**        | 트렌드가 갑자기 뒤집힐 수 있음                     |

저자는 19장에서 "_시장보다 많이 좋은_ 전략은 초과수익을 미래에도 가져갈 가능성이 높다"고 본다. 단, 이는 *과거 결과의 일반화*에 가깝다.

## 5. 17장의 핵심 통찰

> **핵심 통찰 1**: 백테스트 수익률을 *절대값*이 아닌 *5단계 평가*로만 활용. "시장보다 많이 좋다"면 다음 단계.

> **핵심 통찰 2**: 자기 취향이 가장 중요. *NCAV 신봉자가 PER 50인 성장주*를 살 수 없다. 심리적 일관성이 전략 일관성을 만든다.

> **핵심 통찰 3**: 취향이 없다면 *밸류·퀄리티·모멘텀 + 대형주·소형주*에 골고루 분산. 한 전략당 20개 종목.

> **핵심 통찰 4**: 분산 투자로 *수익률 안정성*은 얻을 수 있지만 _MDD는 획기적으로 못 줄인다_. 급락장에서 모든 주식이 같이 떨어지기 때문.

## 요약

- 14~16장의 수십 개 전략 중 무엇을 골라야 하는가의 답은 세 가지: **취향대로 / 전략 분산 / 팩터 모멘텀**.
- *백테스트 수익률*은 절대값이 아니라 _"시장보다 많이 좋다 / 좀 좋다 / 비슷하다 / 별로다 / 매우 안 좋다"_ 5단계 평가용.
- 자신의 *심리적 취향*과 다른 전략은 결국 따르지 못한다. 일관성이 전부.
- 취향 없으면 *밸류 + 퀄리티 + 모멘텀, 대형주 + 소형주*로 분산. 한 전략당 20종목.
- 분산이 _MDD 자체를 낮추지는 못한다_ — 급락장에서 모든 주식의 상관성이 1에 수렴.

## 다른 챕터와의 관계

이 챕터는 *Part 4의 끝*과 *Part 5(그래서 어떻게 투자할까?)*를 잇는 다리다. 14~16장이 "_무엇을_ 살까"였다면, 17장은 "_어떻게 고를까_", 18장은 "_어떻게 위험을 줄일까_"의 흐름.

특히 17장의 마지막 통찰("분산으로 MDD를 못 낮춘다")이 *18장*의 출발점이다. 18장은 *자산 배분*과 *마켓타이밍*으로 MDD 자체를 공격한다.

19장(과최적화 논란)에서는 17장의 "_5단계 평가_"가 *과거 결과를 어디까지 신뢰할지*에 대한 논의로 발전한다.
