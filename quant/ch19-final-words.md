# Chapter 19: 마지막 한마디

## 핵심 질문

이 책에 나온 모든 전략이 _미래에도 통할까_? 팩터가 논문으로 널리 공개되면 *알파*가 사라지지 않을까? *과최적화*인지 어떻게 구분할까? 그리고 가장 어려운 질문 — _왜 좋은 전략을 알면서도 장기적으로 투자자 대부분은 돈을 잃는가_?

이 챕터는 책 전체의 *메타 분석*이다. 14~18장이 "_어떻게 투자할 것인가_"였다면, 19장은 "_과거의 백테스트가 미래에도 의미가 있는가_"라는 근본 질문을 다룬다.

## 1. 과거에 유효했던 전략들이 미래에도 먹힐까? (영상 41)

### 우려와 답

> 책에 소개한 전략의 미래 수익이 주가지수를 지속적으로 능가한다고 _장담하지 못한다_. 또한 우수한 전략이 공개되면 누구나 따라해서 *초과수익이 줄어들 것*이라는 우려가 있을 수 있다.

> **핵심 통찰**: 그러나 과거에 우수했던 전략들이 초과수익을 미래에 *유지할 가능성이 매우 높다*고 본다.

### 야콥스-뮐러 논문 (2018)

독일 두 교수 *Heiko Jacobs*와 *Sebastian Müller*가 *39개국, 241개 투자 팩터*의 수익률을 분석. 핵심: 팩터가 논문으로 공개되기 *전과 후*의 수익을 비교.

### 결과 표 (1980~2015)

| 국가 | 발표 전 (동일가중) | 발표 후 (동일가중) |
| ---- | ------------------ | ------------------ |
| 한국 | 0.55%\*\*\*        | 0.22%              |
| 미국 | 0.56%\*\*\*        | **-0.45%\***       |
| 일본 | 0.22%\*\*\*        | 0.04%              |
| 독일 | 0.51%\*\*\*        | 0.25%\*\*\*        |
| 호주 | 0.74%\*\*\*        | 0.05%              |
| 인도 | 0.58%\*\*\*        | 0.17%              |
| 중국 | 0.22%\*\*\*        | -0.16              |

(\*\*\* 1% 수준 통계 유의)

| 발견                                                             | 의미                             |
| ---------------------------------------------------------------- | -------------------------------- |
| **미국**: 발표 후 수익률 *유의하게 음수*가 됨                    | 헤지펀드가 적극 활용 → 알파 소멸 |
| **나머지 38개국**: 대부분 통계적 유의성 사라지지만 _플러스 유지_ | 알파 일부 유지                   |
| **한국**: 발표 전 0.55% → 발표 후 0.22% (유의 X)                 | 알파 줄지만 여전히 플러스        |

> **핵심 통찰**: 투자 팩터의 공개 여부는 *미국을 제외한 다른 국가*에서는 수익률에 거의 영향을 미치지 않았다. 이것이 저자가 *책과 유튜브를 통해 팩터를 아낌없이 공개*하는 이유.

### 한국에서 공개된 전략이 계속 통하는 이유

| 이유                                                                                        |
| ------------------------------------------------------------------------------------------- |
| **논문 뒤지면서 투자하는 사람이 거의 없음** — 한국 독자들은 거의 못 봄                      |
| **퀀트 전략 실행자가 극소수** — 책 3만 부 중 실제 실행자는 _2% 이하_ 추정                   |
| **기관투자가의 인센티브 부재** — 운용보수가 *수익률이 아닌 자산 규모*에 비례                |
| **펀드매니저는 무난한 종목 선택** — 삼성전자 등 대형주, 다 같이 잃으면 책임 분산            |
| **금융기관 임원의 퀀트 이해 부족** — 듣보잡 중소형주 추천 시 무지 + 책임                    |
| **기관의 소형주 진입 거의 불가능** — 1일 거래량 1억 원 종목에 대규모 자금 투입 시 가격 급등 |

> **핵심 통찰**: 미국과 달리 한국에서는 *소형주 내 퀀트 전략*이 _계속 유효할 가능성이 매우 높다_. 헤지펀드가 수조 달러로 따라잡는 미국과 달리, 한국 기관은 _구조적으로 따라할 수 없다_.

## 2. 퀀트계의 끝없는 논란 — 전략과 과최적화

### 농담 같은 진실

> "내가 만들면 투자전략, 남이 만들면 과최적화"

전략 공개 시 과거 수익률이 높고 MDD가 작은 것이 매력적이지만, 일부는 *일부러 과거 수치에 최적화*해서 그날부터 안 먹히는 사기성 전략도 존재.

### 과최적화 회피 3대 기준

#### 기준 1: 어느 정도 검증된 전략인가?

| 검증                                    |
| --------------------------------------- |
| 동일 전략이 *다른 시장*에서도 통하는가? |
| *다른 시간 구간*에서도 통하는가?        |

> 미국에서 검증된 후 *글로벌 시장*에서 백테스트하는 논문이 많고, 의견이 일치하면 *전 세계적으로 통한다*고 가정 가능.

밸류·퀄리티·모멘텀 — _수십 개국, 수십~수백 년_ 백테스트가 존재.

#### 기준 2: 논리적으로 말이 되는가?

> 2007년 *방글라데시 버터 생산량과 S&P 500 주식 수익률의 상관성이 매우 높다*는 개그성 논문이 나왔다. 물론 말이 안 됨.

| 비교                      | 논리적 근거                                       |
| ------------------------- | ------------------------------------------------- |
| 방글라데시 버터 ↔ S&P 500 | 없음 (우연)                                       |
| 밸류, 퀄리티, 모멘텀      | *인간 심리·편향*과 직결 — 인간이 존재하는 한 지속 |

저자가 책 초반에 투자 심리를 자세히 설명하고, 밸류·퀄리티·모멘텀이 통하는 *이유*를 설명한 것은 이 때문.

#### 기준 3: 비슷한 팩터로 직접 백테스트

> 강환국 울트라 전략은 12개 지표를 활용. _왜 하필 이 12개 지표?_ 라는 의문이 있을 수 있다.

| 검증 방법                                          |
| -------------------------------------------------- |
| 12개 지표가 아닌 *다른 지표*로 변경해도 좋은 성과? |
| PSR을 EV/EBITDA로 바꾸면? GP/A를 ROE로?            |
| 직접 만든 지표(영업이익/매출액 등)는?              |

> **핵심 통찰**: 나무(특정 12개 팩터)가 아니라 *숲(밸류 + 퀄리티 + 모멘텀의 조합)*을 봐야 한다. 다른 지표로 바꿔도 좋은 성과가 나오면 *조합의 위력*을 믿게 된다.

### 학계의 검증

> 강환국 울트라 전략 자체를 백테스트한 논문은 없지만, *비슷한 전략 (밸류 + 퀄리티 + 모멘텀)*을 테스트한 논문은 많다. _"비슷한 전략들이 미국이나 글로벌 시장에서 먹혔는가?"_ 를 찾아보는 것이 실력을 가장 빠르게 높인다.

### 진입 장벽의 변화

```
과거: 데이터 확보 + 엑셀/파이썬/R 백테스트 → 진입장벽 높음
현재: 퀀트킹, 젠포트, 닥터퀀트 → 누구나 백테스트 가능

→ 누구나 자신만의 퀀트 전략을 만들 수 있는 세상이 왔다!
```

## 3. 왜 장기적으로 투자자가 돈을 못 버는가?

### 책의 약속

> 여기까지 읽었으면 종목 선정·MDD 제한 방법을 다 알게 되었다. _"이 책의 전략 선택해서 따라하고 부자 되세요!"_ 라며 책을 마치고 싶다.

> 그런데 실제로 *원금을 10배, 100배로 불린 사람*이 많을까? 아마 거의 못 보았을 것이다. 투자자 대부분은 *중도 하차*한다.

### 의심 마귀의 출현

> 사람은 기본적으로 *의심이 많은 동물*이다. 과거 수익이 화려해도 우리는 과거 수익을 먹을 수 없다. 미래 수익이 나야 돈을 번다. 미래 보장은 누구도 할 수 없다.

### 쓰레기 전략 사례

저자가 두 개의 _"쓰레기 전략"_ 차트를 보여준다.

```
1년~1년 6개월 동안 16% 잃는 전략 (왼쪽)
1년 6개월에 비슷한 손실 (오른쪽)

→ "이 따위 전략에 왜 투자하나?" 누구나 그렇게 생각한다.
```

### 충격적 진실

> **이 쓰레기 전략은 바로 VAA 전략의 일부분이었다!**

VAA(Vigilant Asset Allocation) 전략은 1970~2020년 _연복리 17.8%, MDD 16.1%_ — 50년에 원금이 4,359배가 된 동적 자산배분 전략. 그러나 그 안에 *1~2년의 손실 구간*이 있다.

> **핵심 통찰**: 이 책에 나온 *어떤 아름다운 전략*에도 필연적으로 *"쓰레기 같은" 손실 구간*이 있다. 재수 없으면 1년, 2년, 3년 계속될 수도 있다.

### 50년 시야와 한 달 시야

| 시스템                        | 시야                  |
| ----------------------------- | --------------------- |
| **시스템 2** (이성적, 장기적) | 50년 백테스트, 복리   |
| **시스템 1** (직관적, 단기적) | _하루, 일주일, 한 달_ |

> **핵심 통찰**: 우리는 시스템 2로 *장기적*으로 투자할 수는 있으나, *시스템 1*의 지배를 받는 한 *순간순간은 늘 단기적*이다. 1~6개월 손실이 쌓이면 대부분 지친다.

### 편향에 의한 전략 포기

수만 년에 걸쳐 진화한 *심리 편향*이 우리를 망친다:

| 편향                         | 현상                                                   |
| ---------------------------- | ------------------------------------------------------ |
| **과잉확신**                 | "내가 전략을 조금 바꾸면 더 잘되겠지"                  |
| **전망망상 편향**            | "이런 시국에 주식? 이번 달은 좀 기다리자"              |
| **확증편향**                 | 자신이 보고 싶은 정보만 정확하게 찾아냄                |
| **통계감각 부재 + 사후편향** | "거봐, 11월에 안 사길 잘했지" (한 번 맞춘 것을 일반화) |
| **권위편향**                 | 유명 투자자의 한마디에 흔들림                          |

> 이 모든 편향이 우리를 *훌륭한 전략*에서 멀어지게 한다. 안다고 사라지지 않지만 _모르는 것보다는 낫다_.

### 저자의 변화 — 종교 → 6개월

이전 책 끝에서 저자는 *"전략을 종교처럼 믿고 따르라"*고 했다. 지금은 생각이 바뀌었다.

> 투자 연구는 끝이 없어서 새 기법이 계속 개발된다. 2017년 *강환국 슈퍼가치*가 매우 좋은 줄 알았는데 지금 보니 *강환국 울트라*가 훨씬 좋다. *나에게 맞아 평생 쓸 수 있는 전략*은 영원히 못 찾을 것 같다.

### 저자의 현재 방법 — 6개월 단위

```
9월: 다음 11~4월에 어떤 전략으로 투자할지 깊게 고민
10월 말: 전략 집행 → 4월 말까지 절대 변경 금지

3월: 다음 5~10월에 어떤 전략으로 투자할지 고민
4월 말: 전략 집행 → 10월 말까지 절대 변경 금지
```

> **핵심 통찰**: _6개월 단위로 전략 결정 + 6개월 동안 기계처럼 집행 + 중간 수정 금지_ — 이 방법이 저자에게는 최선.

## 5. 19장의 핵심 통찰

> **핵심 통찰 1**: 팩터가 공개되면 미국에서는 알파가 줄지만 *한국 등 다른 시장*에서는 거의 줄지 않는다 — _야콥스-뮐러 논문_. 한국 기관의 퀀트 미적용·소형주 진입 불가 등 _구조적 이유_.

> **핵심 통찰 2**: 과최적화 회피 3대 기준 — ① *다른 시장·시간*에서도 검증, ② _논리적 설명_ (밸류·퀄리티·모멘텀의 심리학), ③ _비슷한 팩터로 재검증_.

> **핵심 통찰 3**: *나무(특정 12개 팩터)*가 아니라 *숲(밸류+퀄리티+모멘텀)*을 봐라. 지표 바꿔도 결과가 비슷하면 조합의 본질을 믿게 된다.

> **핵심 통찰 4**: 어떤 좋은 전략에도 *"쓰레기 구간"*이 필연적이다. VAA(연복리 17.8%) 안에도 1~2년 손실 구간이 있다.

> **핵심 통찰 5**: 시스템 1(직관적, 단기적)이 시스템 2(이성적, 장기적)를 이긴다 — 단기 부진 → *과잉확신·전망망상·확증편향·권위편향*에 빠져 전략 포기.

> **핵심 통찰 6**: 저자의 새 방법 — _6개월 단위로 전략 결정 + 6개월 동안 절대 수정 금지_. 종교적 추종이 아닌 _주기적 재검토 + 단기 일관성_.

> **핵심 통찰 7**: _"좋은 전략 + 일관성 + 임의 개입 금지"_ — 이 셋만 지키면 머지않아 경제적 자유에 도달한다.

## 요약

- 팩터 공개 후 미국은 알파 소멸하지만 *한국 등 38개국*은 거의 그대로 — 한국 기관의 구조적 한계 덕분.
- 과최적화 회피: ① _다른 시장·시간 검증_, ② _논리적 설명_ (인간 심리), ③ _비슷한 지표 재검증_.
- 나무(특정 지표)가 아닌 _숲(팩터 조합)_ 보기.
- 모든 좋은 전략에 *쓰레기 구간(1~3년 손실)*이 있다. VAA(50년 17.8%) 안에도 1.5년 손실 구간 존재.
- *시스템 1(단기 직관)*이 *시스템 2(장기 이성)*를 이긴다 — 1~6개월 부진 시 *심리 편향*들로 전략 포기.
- 저자의 변화: _"종교처럼 믿어라"_ → _"6개월 단위 + 절대 수정 금지"_.
- 결국 모든 것은 *좋은 전략의 실현*과 _일관성_. 임의 개입만 안 하면 경제적 자유.

## 다른 챕터와의 관계

이 챕터는 **책 전체의 마무리**다. 14~18장이 *어떻게 투자할까*였다면 19장은 *왜 그래도 대부분 실패하는가*에 대한 답.

본 챕터의 *야콥스-뮐러 논문*은 책 전체의 *근거*다 — 14~16장의 모든 전략이 한국에서 *공개 후에도 통할 가능성*이 높다는 학술적 뒷받침.

*과최적화 3대 기준*은 14~16장의 백테스트를 다시 검증하는 _체크리스트_. 강환국 울트라 전략도 이 기준으로 _재검증_ 가능.

*시스템 1 vs 시스템 2*의 통찰은 18장의 _MDD 20% 이하_ 원칙과 짝을 이룬다 — *손실 구간을 견디는 능력*이 결국 모든 것을 결정한다.

저자의 *6개월 룰*은 17장의 *전략 선택*과 18장의 *마켓타이밍*을 통합하는 실전 운영 룰이다.

> 결국 모든 것은 *좋은 투자 전략의 실현*과 *일관성*에 달렸다. 그 전략을 중간에 포기하지 않으면, 또는 임의로 개입하지 않으면, 전략이 어느 정도만 괜찮아도 여러분은 머지않은 시간에 *경제적 자유*에 도달할 것이다.
