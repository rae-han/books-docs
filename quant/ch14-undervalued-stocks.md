# Chapter 14: 초저평가주를 찾아서

## 핵심 질문

벤저민 그레이엄의 *NCAV(Net Current Asset Value)*와 강환국 *슈퍼가치 전략*을 어떻게 더 강력하게 업그레이드할 수 있는가? 14년~20년 백테스트에서 연복리 30~50%를 내는 가치 전략의 비결은 무엇인가?

이 챕터는 *Part 4(새로운 투자 전략)*의 첫 챕터로, 전작 『할 수 있다! 퀀트투자』(2017)에서 소개한 **밸류 전략**을 새로운 데이터·도구로 업그레이드한다. 백테스트 도구는 기존 엑셀에 더해 *퀀트킹*과 *뉴지스탁 젠포트*를 활용한다.

> **핵심 통찰**: 팩터는 *재료*이고 전략은 *요리*다. 14장은 지금까지 확보한 재료(밸류·퀄리티·모멘텀 팩터)로 새로운 요리를 시작하는 챕터다.

## 1. 지금까지의 팩터 정리

저자가 책의 1~3부에서 다룬 팩터들이다.

| 분류       | 팩터                                                                                                                             |
| ---------- | -------------------------------------------------------------------------------------------------------------------------------- |
| **밸류**   | PER, PBR, PCR, PSR, NCAV, EV/EBIT, PFCR                                                                                          |
| **퀄리티** | GP/A, 영업이익/자본총계, ROE, ROA, F-스코어, 자산성장률, 차입금비율 증가율, 영업이익/차입금 증가율, 이익변동성, 주가변동성, 베타 |
| **모멘텀** | 주가모멘텀(1개월 역추세, 3~12개월 추세), 영업이익 모멘텀, 순이익 모멘텀                                                          |
| **기타**   | 11~4월 투자 (계절성)                                                                                                             |

> **핵심 통찰**: 백테스트의 공통 가정 — 슬리피지 없음, 매도수수료 없음, 2007/06~2021/05의 14년 구간, 편입 종목 20개로 통일.

## 2. NCAV 전략 업그레이드 (영상 443)

### 오리지널 NCAV (벤저민 그레이엄)

_NCAV_: 청산가치 / 시가총액. 가장 높은 20개 종목을 매수.

| 항목           | NCAV 20개 종목 |
| -------------- | -------------- |
| 누적 수익률    | 942.8%         |
| 연 평균 수익률 | **17.4%**      |
| MDD            | 56.5%          |
| 승률           | 61.3%          |

### 업그레이드 1: 금융사·지주사·관리종목 제외

> 저자 경험: NCAV 전략을 오래 실행해 보니 *지주사와 금융사*에 _밸류 트랩(value trap)_ — 매우 저평가되어 보이지만 주가가 오르지 않고 영원히 저평가되는 기업 — 이 많았다. 중국 기업은 회계 신뢰성이 낮아(영상 537), 스팩은 페이퍼 컴퍼니라서 제외.

| 항목           | 업그레이드 1       |
| -------------- | ------------------ |
| 연 평균 수익률 | **22.4%** (+5.0%p) |
| MDD            | 53%                |
| 승률           | 63.1%              |

### 업그레이드 2: + GP/A 상위 50% + 차입금비율 200% 이하

"저평가됐고 _실제로 약한 기업_"을 빼고 "보통 수준인데 저평가된 기업"에만 투자.

| 항목           | 업그레이드 2       |
| -------------- | ------------------ |
| 연 평균 수익률 | **23.5%** (+1.1%p) |
| MDD            | 53.3%              |
| 승률           | 64.9%              |

> **핵심 통찰**: 업그레이드 1과 큰 차이는 없지만 "더 안전하고 수익성도 있는 기업에 투자한다"는 *심리적 안정감*을 얻을 수 있다. 필터는 취향이다.

### 업그레이드 3: + 11~4월만 전략 실행

| 항목           | 업그레이드 3                            |
| -------------- | --------------------------------------- |
| 연 평균 수익률 | 20.2% (-3.3%p)                          |
| **MDD**        | **25.3%** (53% → 25.3%, 절반 이상 감소) |
| 승률           | 71.4%                                   |

연복리는 다소 줄지만 _MDD가 절반 이상 줄었다_. 5~10월에는 채권 등 저위험 자산 보유로 어느 정도 수익을 더 낼 수 있어, 실질적으로 6개월만 주식에 투자해도 업그레이드 2와 거의 같은 수익률에 MDD 25%로 유지 가능.

### NCAV 업그레이드 종합

```
오리지널        : 17.4% / MDD 56.5%
↓ 금융사·지주사·관리종목 제외
업그레이드 1    : 22.4% / MDD 53.0%
↓ + GP/A 상위 50% + 차입금비율 200% 이하
업그레이드 2    : 23.5% / MDD 53.3%
↓ + 11~4월만 투자
업그레이드 3    : 20.2% / MDD 25.3%   ◀ 권장
```

## 3. 강환국 슈퍼가치 전략 업그레이드 (영상 439)

### 오리지널 강환국 슈퍼가치 (Ch 8)

분기 PER, PBR, 분기 PCR, 분기 PSR — _4개 지표 평균 순위_ 상위 20개 매수.

| 항목           | 결괏값    |
| -------------- | --------- |
| 누적 수익률    | 8067.8%   |
| 연 평균 수익률 | **36.8%** |
| MDD            | 43.4%     |
| 승률           | 65.5%     |

### 한국 밸류 지표 조합 백테스트 (2007/03~2021/02)

여러 밸류 지표 조합을 시험한 결과:

| 포함 지표                                           | 연복리    | MDD       | 승률      |
| --------------------------------------------------- | --------- | --------- | --------- |
| 분기 PER, PBR                                       | 26.2%     | 49.7%     | 64.9%     |
| 분기 PER, 분기 PSR                                  | 30.4%     | 46.3%     | 60.7%     |
| 분기 PER, 분기 PFCR                                 | 36.1%     | 47.9%     | 64.3%     |
| **분기 PER, PBR, 분기 PCR, 분기 PSR (오리지널)**    | **36.8%** | **43.4%** | **65.5%** |
| **분기 PER, PBR, 분기 PFCR, 분기 PSR (업그레이드)** | **38.3%** | **42.0%** | **66.1%** |
| 분기 PER, PBR, 분기 PCR, 분기 PFCR, 분기 PSR        | 36.7%     | 42.7%     | 67.3%     |

### 백테스트의 두 가지 결론

> **핵심 통찰 1**: 밸류 지표는 아무거나 _2개만 합쳐도_ 최근 14년간 복리 20% 이상에 도달한다.

> **핵심 통찰 2**: 지표를 3~6개 사용한다고 2개보다 늘 우수한 것은 아니다. 어떤 지표를 사용할지·몇 개 사용할지 고민할 필요는 없다.

### 저자의 권장 — PCR을 PFCR로

| 분류          | 매수 전략                                                                             |
| ------------- | ------------------------------------------------------------------------------------- |
| **분류**      | 밸류                                                                                  |
| **매수**      | 1/PER, 1/PBR, 1/PFCR, 1/PSR — 4개 지표 평균 순위 상위 매수 (모두 최근 분기 실적 기준) |
| **매수 종목** | 20개 권장 (30, 50개도 무난)                                                           |
| **매도**      | 4월·10월 마지막 거래일 (연 2회 리밸런싱)                                              |

> **핵심 통찰**: 영업활동에서 벌어들이는 현금흐름(_PCR_)보다는 설비투자를 끝내고 *주주들에게 돌아가는 잉여현금흐름(PFCR)*이 더 중요하다. 단, PCR을 써도 별 문제는 없다.

### 업그레이드 슈퍼가치 십분위수 (2001/04~2020/12)

| 분위         | 연중 (%)  | 11~4월 (%) | 5~10월 (%) |
| ------------ | --------- | ---------- | ---------- |
| **1 (최고)** | **23.45** | 18.59      | 5.25       |
| 2            | 18.59     | 17.08      | 2.19       |
| 5            | 9.16      | 13.15      | -3.08      |
| 10 (최저)    | -17.98    | -2.82      | -16.48     |
| 1-10 차이    | **41.43** | 21.41      | 21.73      |

> **핵심 통찰**: 1분위(상위 10%)와 10분위(하위 10%)의 차이가 41%p. 분위가 깔끔하게 계단을 이루어 *팩터의 신뢰성*이 입증된다.

## 4. 슈퍼가치 + 소형주 + 계절성

### 11~4월만 투자

| 항목           | 결괏값 |
| -------------- | ------ |
| 연 평균 수익률 | 23.6%  |
| MDD            | 34.2%  |
| 승률           | 69%    |

> **핵심 통찰**: 강환국 슈퍼가치 전략은 _예외적으로_ 11~4월만 투자하기보다는 1년 내내 투자하는 게 낫다. 다른 전략과 달리 5~10월에도 양호한 편이다. 단, 과거 결과일 뿐 미래 보장은 없다.

### 시가총액 하위 20% (소형주)

| 항목           | 결괏값    |
| -------------- | --------- |
| 누적 수익률    | 23,227.1% |
| 연 평균 수익률 | **47.6%** |
| MDD            | 45.6%     |
| 승률           | 68.5%     |

연복리가 30%대에서 *40%대 후반*으로 점프. 소형주에서 강환국 슈퍼가치 전략의 위력이 극대화된다.

### 소형주 + 11~4월

| 항목           | 결괏값    |
| -------------- | --------- |
| 누적 수익률    | 4067.3%   |
| 연 평균 수익률 | **30.3%** |
| MDD            | **27.9%** |
| 승률           | 71.4%     |

> **핵심 통찰**: 소형주 전략을 11~4월에만 실행하면 연복리는 47.6% → 30.3%로 줄지만, MDD는 45.6% → 27.9%로 크게 감소. _수익 대비 위험_ 측면에서 가장 무난한 선택지.

## 6. 14장의 핵심 통찰

> **핵심 통찰 1**: 한국 시장에서 밸류 팩터는 _14년 백테스트_ 기준으로 매우 강력하다. 가장 단순한 NCAV조차 연복리 17.4%, 슈퍼가치는 36.8%다.

> **핵심 통찰 2**: 필터를 추가할수록 수익률 상승은 *체감*된다. 가장 큰 차이는 *11~4월 투자*가 만든다 — MDD를 절반 수준으로 줄인다.

> **핵심 통찰 3**: 소형주(시가총액 하위 20%)는 한국 시장에서 압도적 초과수익을 만든다. 단, MDD는 거의 줄지 않는다.

> **핵심 통찰 4**: _PFCR > PCR_ (영업현금흐름보다 잉여현금흐름이 주주에게 더 의미 있음). 단, 큰 차이는 아니다.

## 요약

- 14장은 _Part 4_ 도입부로, 기존 *NCAV*와 _강환국 슈퍼가치_ 전략을 업그레이드한다.
- **NCAV 업그레이드**: 금융사·지주사·관리종목 제외 + GP/A 상위 50% + 차입금비율 200% 이하 → 17.4% → 23.5% (MDD 53% 유지). 11~4월만 투자 시 MDD 25.3%로 급감.
- **강환국 슈퍼가치 업그레이드**: PCR 대신 _PFCR_ 사용 권장. 연복리 36.8% → 38.3%.
- 밸류 지표는 *아무거나 2개*만 합쳐도 14년 복리 20% 이상.
- *지표 3~6개*가 2개보다 늘 우수하지는 않다.
- **소형주(시가총액 하위 20%)**에서 슈퍼가치 전략은 *연복리 47.6%*로 폭발한다.
- 슈퍼가치는 _예외적으로_ 1년 내내 투자가 11~4월만 하기보다 낫다.
- 모든 전략의 표준 매도 규칙: **4월·10월 마지막 거래일 리밸런싱 (연 2회)**.

## 다른 챕터와의 관계

이 챕터는 *Part 4(새로운 투자 전략)*의 시작이다. 다음 *15장*은 같은 개념을 **퀄리티 팩터**에 적용하고, *16장*은 모멘텀 팩터까지 통합한 *강환국 울트라 전략*으로 연결된다.

이 챕터의 모든 백테스트는 _공통 가정_(20개 종목, 4월·10월 리밸런싱, 슬리피지 무시, 14년 또는 20년 구간)을 따른다. 이 가정은 책 전체에 적용된다.

*18장(MDD를 줄이는 방법)*에서는 본 챕터의 11~4월 투자를 더 발전시킨 *마켓타이밍*과 _자산배분 + 퀀트_ 기법이 등장한다.
