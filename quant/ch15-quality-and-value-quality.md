# Chapter 15: 우량주 + 저평가 우량주 전략

## 핵심 질문

밸류 팩터가 검증되었으니, *퀄리티(우량주) 팩터*는 어떻게 활용할 수 있는가? 2017년 강환국 슈퍼퀄리티 전략을 새 데이터로 다시 백테스트하면 어떤 결과가 나올까? 파마(Fama)의 노벨상급 연구를 종합한 *최종병기*는? 배당 전략의 핵심 지표 *배당성향*은 어떻게 활용할까? 그리고 *밸류와 퀄리티의 시너지*는?

이 챕터는 14장의 밸류 전략에 이어 **퀄리티 팩터**를 본격적으로 다루며, 마지막에 *밸류 + 퀄리티 통합 전략*까지 발전시킨다.

## 1. 강환국 슈퍼퀄리티 전략 (영상 503)

### 2017년 오리지널의 좌절

전작 *『할 수 있다! 퀀트투자』*의 강환국 슈퍼퀄리티 전략은 _신F-스코어 3점 + GP/A 상위_ 기업을 매수했다. 새 백테스트(2000/10~2020/12)에서 다시 검증한 결과:

| 신 F-스코어 3점 + GP/A 분위 | 연중 (%) | 11~4월 (%) | 5~10월 (%) |
| --------------------------- | -------- | ---------- | ---------- |
| 1 (최고)                    | 14.14    | 12.97      | 1.37       |
| 2                           | 16.50    | 14.41      | 2.22       |
| 6                           | 17.40    | 15.92      | 1.69       |
| 10 (최저)                   | 11.60    | 10.95      | 0.86       |
| **1-10 차이**               | **2.54** | 2.02       | 0.51       |

> **핵심 통찰**: 1분위와 10분위 차이가 2.54%p에 불과 — _팩터로서 실패_. 신 F-스코어 3점 기업들은 평균 17%의 수익률을 내지만, 그 안에서 GP/A로 다시 구분해도 초과수익이 나오지 않았다.

> 저자 자조: "이래서 직접 백테스트하는 것이 중요하고, 심지어 내가 한 백테스트도 곧이곧대로 믿으면 안 되고 계속 검증해야 한다."

유진 파마 교수에게 배운 것을 모두 종합한 전략:

|               | • 1/PBR (내림차순) — **50%**                                                   |

## 3. 배당 전략 (영상 458)

### 배당주의 우위

> 한국 상장기업들은 보통 3~4월에 배당을 지급. "따르릉!" 하며 배당금 수십만 원이 들어왔다는 핸드폰 문자가 오면 가슴이 따뜻하고 영혼이 행복해진다.

| 구분      | 연복리 수익률 (%) (2000/10~2020/12) |
| --------- | ----------------------------------- |
| 배당 있음 | 11.33%                              |
| 배당 없음 | 5.81%                               |

> **핵심 통찰**: 배당주 자체가 *연복리 5.5%p의 초과수익*을 낸다.

### 배당 수익률 정의

```
배당 수익률 = 최근 12개월 1주당 배당금 / 주가
```

10,000원 주식이 12개월간 배당 500원 → 배당수익률 5%.

### 배당 수익률 십분위수

| 분위         | 연중 (%)  |
| ------------ | --------- |
| **1 (최고)** | **15.50** |
| 2            | 16.27     |
| 5            | 14.42     |
| 10 (최저)    | 2.47      |
| 1-10 차이    | **12.03** |

배당수익률이 높은 기업의 주식 수익률이 낮은 기업보다 _연복리 12%p_ 더 높다.

### 배당성향의 함정과 해결

```
배당성향 = 배당 / 순이익
```

| 사례                                     | 의미               |
| ---------------------------------------- | ------------------ |
| 순이익 100억, 배당 200억 (배당성향 200%) | 유지 불가능        |
| 순이익 100억, 배당 20억 (배당성향 20%)   | 안전하게 유지 가능 |

너무 낮으면 _"배당주"라 부르기 어렵고_, 너무 높으면 _유지 불가능_.

### 배당 전략 룰

| 분류          | 매수 전략                                                                  |
| ------------- | -------------------------------------------------------------------------- |
| **분류**      | 밸류 + 퀄리티                                                              |
| **매수**      | 전체 주식 중 배당수익률이 가장 높은 주식 매수, 단 _배당성향 30~75%_ 기업만 |
| **매수 종목** | 20개 권장                                                                  |
| **매도**      | 4월·10월 마지막 거래일 (연 2회)                                            |

### 배당성향 30~75% + 배당수익률 오분위 (2000/10~2020/12)

| 분위         | 연중 (%)  |
| ------------ | --------- |

| 배당성향 하단 | 배당성향 상단 | 상위 20% 수익률 |

## 6. 15장의 핵심 통찰

> **핵심 통찰 1**: 한 번 검증된 전략도 _재검증해야 한다_. 2017년 슈퍼퀄리티 전략은 새 데이터에서 *실패*했다. 4년 데이터의 우연을 진실로 받아들였던 것.

> **핵심 통찰 2**: _4팩터 결합_ 슈퍼퀄리티(GP/A + 영업이익/차입금 증가율 + 자산성장률 + 주가변동성)는 단일 팩터보다 6배 이상 강력하다.

> **핵심 통찰 3**: 파마의 노벨상 이론을 _밸류 50% + GP/A 25% + 자산성장률 25%_ 가중치로 종합하면 연복리 32.43%.

> **핵심 통찰 4**: 배당 전략의 핵심은 _배당성향_ 필터 — "유지 가능한 배당주"만 골라내야 한다.

> **핵심 통찰 5**: 밸류와 퀄리티의 결합은 시너지 — 21~23%대 → 28%대로 점프, 소형주에서는 38%대로.

## 요약

- 2017년 슈퍼퀄리티 전략은 새 데이터에서 _실패_. 백테스트도 재검증해야 한다.
- **업그레이드 슈퍼퀄리티 전략**: 신 F-스코어 3점 + GP/A·영업이익/차입금 증가율·자산성장률·주가변동성 4개 평균 순위. 연복리 21.2%, 소형주 31.9%.
- **파마의 최종병기**: 1/PBR 50% + GP/A 25% + 자산성장률 25% 가중치 + 소형주 한정. 연복리 32.4%.
- **배당 전략**: 배당성향 30~75%인 기업 중 배당수익률 상위. 연복리 19.6%. 배당성향 범위는 _대동소이_ — 과최적화 아님.
- **슈퍼가치 + 퀄리티 통합**: 8개 팩터 평균 순위. 연복리 28.2%, 소형주 38.7%.
- 모든 전략의 매도: _4월·10월 마지막 거래일_ 연 2회 리밸런싱.

## 다른 챕터와의 관계

이 챕터는 *14장(밸류)*에 이어 *퀄리티*와 *밸류+퀄리티*를 다룬다. 다음 *16장*에서는 마지막 팩터인 **모멘텀**을 결합해 _밸류+모멘텀, 퀄리티+모멘텀, 강환국 울트라(밸류+퀄리티+모멘텀)_ 전략으로 발전한다.

15장의 모든 전략은 14장의 _4월·10월 리밸런싱_ 규칙을 그대로 이어받는다. 이 일관성 덕분에 17~18장에서 *전략 선택과 분산*을 다룰 때 같은 기준으로 비교할 수 있다.

15장의 _재검증의 교훈_("백테스트도 의심하라")은 *19장(과최적화 논란)*과 직결된다.
