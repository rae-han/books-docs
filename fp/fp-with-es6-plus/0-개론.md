# 0. 개론

## 1.1. 평가
코드가 계산(Evaluation) 되어 값을 만드는 것
```typescript
1; // 1
1 + 2; // 3
[1, 2]; // [1, 2]
[1, 2 + 3]; // [1, 5]
```

## 1.2. 일급
- 값으로 다룰 수 있다.
- 변수에 담을 수 있다.
- 함수의 인자로 사용될 수 있다.
- 함수의 결과로 사용될 수 있다.

## 1.3. 일급 함수
- 함수를 값으로 다룰 수 있다.
- 조합성과 추상화의 도구.

## 1.4. 고차 함수
함수를 값으로 다루는 함수. 고차 함수의 종류는 다음과 같다.

1. 함수를 값으로 다룰 수 있다 → 변수에 함수를 값으로 담을 수 있다.
2. 함수가 일급이다 → 함수의 결과 값으로 함수가 사용될 수 있다.

### 1.4.1. 함수를 인자로 받아서 실행하는 함수
함수를 값으로 받고 다른 값을 받아 안에서 함수에 인자를 적용하는 방법으로 어플리케이티드 프로그램이라고도 한다.
```typescript
const apply10 = f => f(10);
const add5 = num => num + 5;

log(apply10(add5)) // 15

const times = (f, n) => {
  let i = -1;
  while( ++i < n) f(i)
}

times(log, 3) // 0 1 2
times((v) => log(v + 10), 3); // 10 11 12
```

### 1.4.2. 함수를 만들어 리턴하는 함수 (클로저를 만들어 리턴하는 함수)
함수가 만들어 질때의 환경인 x와 함수 자체 객체를 통칭하는 용어이다. 아래 코드에서 addMaker는 클로저를 리턴한다.
```typescript
const addMake = x => y => x + y;
const add10 = addMake(10);

log(add10(5)) // 15
log(add10(10)) // 20
```
