# 이것이 취업을 위한 코딩 테스트다 (with 파이썬)

> 『이것이 취업을 위한 코딩 테스트다 with 파이썬』 (나동빈 지음, 한빛미디어, 2020)
> 이 노트는 모든 코드 예제를 **TypeScript**로 옮긴 변환 학습 노트

원서는 **Python** 기반이지만, 이 노트는 프론트엔드/Node.js 개발자가 바로 활용할 수 있도록 모든 예제를 TypeScript로 정리했다. Python 특유의 도구(`heapq`, `bisect`, `deque`, 슬라이싱)를 TS 관용구로 대체하는 **변환 포인트**를 함께 담았다.

## 개요

이론(Part 2) → 유형별 기출(Part 3)의 구조로, 이론 챕터에서 만든 도구(MinHeap, Union-Find, 조합·순열 헬퍼)가 기출 풀이에 그대로 재사용된다.

TS 고유의 함정 — `Array.prototype.sort()`의 사전순 기본 동작, 우선순위 큐 부재(MinHeap 직접 구현), 중간점 비트 시프트 오버플로, `Math.trunc` vs `Math.floor`, `shift()`의 O(n) — 을 콜아웃으로 강조한 것이 이 노트의 차별점이다.

> 원서 Ch01(개요·복잡도)·Ch02(유형 분석)는 ch00과 중복되어 생략, 부록 B(기타 알고리즘)·C(개발형)·D(기출 풀이)는 노트 범위에서 제외했다.

## 핵심 개념 맵

- **유형 분류가 절반**: 그리디/구현/탐색/정렬/이진 탐색/DP/최단 경로/그래프 — 문제를 유형에 매핑하는 눈이 핵심
- **탐색의 뼈대는 DFS(스택·재귀) vs BFS(큐)** — 격자 문제는 그래프 모델링부터
- **파라메트릭 서치**: "조건을 만족하는 최댓값" 문제를 이진 탐색으로 뒤집기
- **DP = 점화식 + 저장**: 탑다운(메모이제이션) vs 바텀업(테이블)
- **TS 변환 포인트가 실전 함정**: sort 사전순·MinHeap 부재·shift() O(n) — Python과 다른 지점을 알아야 통과한다
