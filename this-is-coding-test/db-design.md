## DB 외 메모

DB 항목으로 두지 않지만 참고용으로 남겨두는 부록.

- **부록 B — 기타 알고리즘**: 소수 판별, 에라토스테네스의 체, 순열/조합, 최대공약수(GCD). 코테에 보조 도구로 자주 등장하지만 별도 챕터로 두지 않음. 문제 풀이 중 필요할 때 참고.
- **부록 C — 개발형 코딩 테스트**: REST API, JSON, requests 라이브러리. 우선순위 낮음. 학습 대상에서 제외.
