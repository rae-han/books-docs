# 코딩 테스트 합격자 되기: 자바스크립트 편

> 『코딩 테스트 합격자 되기 — 자바스크립트 편』 (이선협·박경록 지음, 골든래빗, 2024)

자료구조·알고리즘과 프로그래머스 빈출 100문제로 코딩 테스트를 대비하는 책. 자료구조 단위로 개념 → 몸풀기 → 모의테스트가 반복되는 실전형 구성이다.

| **대상 독자** | JS로 코딩 테스트를 준비하는 취준·이직 개발자 |

## 개요

첫째 마당이 준비(효율적 준비법·프로그래머스 활용·복잡도·필수 문법), 둘째 마당이 자료구조/알고리즘 유형별 정복(배열~그리디 12개 장)이다. 문제가 프로그래머스 실제 기출 중심이라 플랫폼 감각까지 함께 잡힌다.

[this-is-coding-test/](../this-is-coding-test/)(나동빈, TS 변환 노트)와 상보적 — 그쪽이 알고리즘 유형 이론 중심이라면, 이 책은 자료구조 축 + 프로그래머스 문제 풀이량 중심이다.

## 학습 가이드

1. **첫째 마당(Ch00~04)은 빠르게** — Ch03(복잡도)·Ch04(필수 문법)만 정독해도 충분
2. **둘째 마당은 자료구조(Ch05~10) → 알고리즘(Ch11~16) 순서로** — 뒤 장들이 앞 자료구조를 재료로 쓴다
3. **각 장은 개념 → 몸풀기 → 모의테스트 순서 유지** — 몸풀기를 건너뛰면 모의테스트 난이도가 급격해진다
4. **시험 직전엔 handbook.md** — 40쪽 압축 정리로 전체 복습
5. 나동빈 책 노트([this-is-coding-test/](../this-is-coding-test/))와 병행 시 유형 이론은 그쪽, 문제 풀이량은 이쪽

## 핵심 개념 맵

- **자료구조가 축**: 배열→스택/큐→해시→트리→집합→그래프 순서로 재료를 쌓고, 백트래킹~그리디에서 조합한다
- **복잡도 먼저**: 제약 조건(n 크기)에서 허용 복잡도를 역산하는 습관 (Ch03)
- **JS 코테 함정**: `sort()` 사전순, `shift()` O(n), 재귀 깊이 — 언어 특성이 정답률을 가른다
- **문제 풀이 5단계**: 문제 설명 → 접근 아이디어 → 트레이스 → 코드 → 복잡도 — 모든 문제에 동일 적용

## 시그니처 요소와 표기 규칙

- 챕터 구조: `핵심 질문` → 개념 → `시간 복잡도` → `핵심 코드 패턴` → `JavaScript 빌트인 메서드 레퍼런스` → `몸풀기 문제` → `합격자가 되는 모의테스트` → `함정/실수 노트` → `요약`
- 문제 풀이 형식: 문제 설명 → 접근 아이디어 → 알고리즘 트레이스(필요 시) → 코드(주석 포함) → 시간/공간 복잡도
- 코드는 책 원본대로 **JavaScript**
