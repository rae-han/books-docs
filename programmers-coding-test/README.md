## 학습 가이드

각 챕터는 다음 구조로 정리되어 있다.

1. `## 핵심 질문`: 챕터를 관통하는 물음
2. `## N. 자료구조/알고리즘 개념`: 이론 설명과 시각화
3. `## 시간 복잡도`: 빅오 표기 기준 연산별 비용
4. `## 핵심 코드 패턴`: 자주 쓰는 구현 템플릿
5. `## JavaScript 빌트인 메서드 레퍼런스`: 챕터에서 자주 쓰는 메서드 정리
6. `## 몸풀기 문제`: 저자 출제 워밍업 문제 풀이
7. `## 합격자가 되는 모의테스트`: 프로그래머스 기출 문제 풀이
8. `## 함정/실수 노트`: 자주 틀리는 포인트
9. `## 요약`: 챕터 핵심 정리

## 코드 표기

코드 예제는 책 원본대로 **JavaScript**로 작성한다.

## 문제 풀이 형식

각 문제는 다음 순서로 정리한다.

1. **문제 설명**: 입출력, 제약사항 요약
2. **접근 아이디어**: 어떤 자료구조/알고리즘을 왜 선택했는가
3. **알고리즘 트레이스**: 단계별 동작 시각화 (필요 시)
4. **코드**: 주석 포함 풀이
5. **시간/공간 복잡도**
